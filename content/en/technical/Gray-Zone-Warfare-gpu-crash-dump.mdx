<Section icon={Lightbulb} title="Step-by-Step Troubleshooting for GPU Crashes">

Follow these steps to systematically diagnose and resolve GPU crash dump issues in Gray Zone Warfare. Each step aims to eliminate common causes of graphics card instability.

<Steps>
  <Step num="1" title="Update Graphics Drivers">
    Outdated or corrupted graphics drivers are the most frequent culprits behind GPU crashes. Ensure your drivers are up-to-date.
    - **NVIDIA Users**: Download the latest Game Ready Driver from the official NVIDIA website. Perform a clean installation using DDU (Display Driver Uninstaller) for best results.
    - **AMD Users**: Download the latest Adrenalin Software from the official AMD website. Opt for a factory reset during installation.
    - **Intel Users**: Download the latest drivers from the official Intel support page.
    <Callout type="tip">
    Always download drivers directly from the manufacturer's official website. Beta drivers might offer new features but can introduce instability. Stick to stable releases for gaming.
    </Callout>
  </Step>
  <Step num="2" title="Adjust In-Game Graphics Settings">
    High graphics settings can overstress your GPU, especially during intense combat or scene transitions.
    - **Reduce Texture Quality**: Lowering this setting significantly reduces VRAM usage.
    - **Lower Shadow Quality**: Shadows are resource-intensive; reducing their quality can free up GPU cycles.
    - **Disable Ray Tracing (if applicable)**: Ray tracing is extremely demanding and can cause instability on some systems.
    - **Cap Frame Rate**: Limit your FPS to a stable rate, ideally below your monitor's refresh rate, to prevent the GPU from working harder than necessary.
    - **Use a Lower Resolution**: If other steps fail, try playing at a slightly lower resolution.
  </Step>
  <Step num="3" title="Monitor System Temperatures">
    Overheating components can lead to throttling and crashes.
    - Use monitoring software like MSI Afterburner or HWMonitor to check your GPU and CPU temperatures during gameplay.
    - Ensure good airflow in your PC case. Clean dust from fans and heatsinks.
    - Consider reapplying thermal paste to your GPU or CPU if temperatures remain high despite good airflow.
    <Callout type="info">
    Optimal GPU temperatures during heavy load are typically below 80°C (176°F). Consistently higher temperatures indicate a cooling issue.
    </Callout>
  </Step>
  <Step num="4" title="Verify Game File Integrity">
    Corrupted game files can also lead to unexpected crashes.
    - **Steam**: Right-click Gray Zone Warfare in your library > Properties > Installed Files > Verify integrity of game files.
    - This process will check for and replace any damaged or missing game files.
  </Step>
  <Step num="5" title="Disable Overclocks and Undervolting">
    While overclocks can boost performance, even slight instability can cause crashes in demanding games.
    - Revert your GPU and CPU to their stock clock speeds.
    - If you have undervolted your GPU, reset it to default settings.
    - Test the game with default settings to see if stability improves. If it does, gradually reapply overclocks with thorough testing.
  </Step>
</Steps>

<Section icon={Shield} title="Advanced Solutions and System Optimization">

If the basic troubleshooting steps don't resolve your GPU crash dump issues, consider these more advanced solutions and system optimizations.

<CardGrid cols={2}>
  <Card title="Update Windows">
    Ensure your Windows operating system is fully updated. Microsoft frequently releases updates that include performance improvements and bug fixes that can impact game stability.
    - Go to `Settings > Windows Update` and check for updates.
    - Install all pending updates and restart your system.
  </Card>
  <Card title="Adjust Power Management Settings">
    Sometimes, insufficient power delivery can cause GPU instability.
    - In Windows, go to `Control Panel > Power Options`.
    - Select "High performance" or "Ultimate performance" plan.
    - Ensure PCI Express Link State Power Management is set to "Off" in advanced power settings.
  </Card>
  <Card title="Increase Virtual Memory">
    If your system runs out of physical RAM, it relies on virtual memory (page file). Insufficient virtual memory can contribute to crashes.
    - Go to `System Properties > Advanced > Performance Settings > Advanced > Virtual memory > Change`.
    - Uncheck "Automatically manage paging file size" and set a custom size, usually 1.5-2 times your RAM.
  </Card>
  <Card title="Check PSU Wattage">
    An underpowered Power Supply Unit (PSU) can lead to system instability, especially under heavy load.
    - Ensure your PSU meets the recommended wattage for your GPU and other components.
    - If unsure, use an online PSU calculator to estimate your system's power needs.
  </Card>
</CardGrid>
