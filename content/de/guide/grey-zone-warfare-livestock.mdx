## Aufgabenübersicht und Anforderungen

Bevor Sie ins Feld ziehen, ist es wichtig zu verstehen, was Banshee von Ihnen erwartet. Diese Aufgabe ist keine einfache „Search and Destroy“-Mission; sie beinhaltet Interaktion mit der Umgebung und Rätsellösungen. Sie werden in einer Zone operieren, die normalerweise von KI-Kämpfern der mittleren bis oberen Preisklasse bevölkert ist. Daher ist es nicht verhandelbar, mit der richtigen Ausrüstung vorbereitet zu sein.

## Schritt 1: Lokalisierung der Käfige und der Hütte

Die erste Phase der **Grey Zone Warfare Livestock**-Mission erfordert, dass Sie den primären Ort des Interesses identifizieren. Ihr Ziel ist eine kleine Holzhütte in der Nähe einer Reihe von Tierkäfigen. Dieser Ort befindet sich in der südwestlichen Ecke des Autowerkstatt-Komplexes.

Sobald Sie an der markierten Stelle auf Ihrer Karte angekommen sind, suchen Sie nach dem Außenzaun der Werkstatt. Anstatt durch das Haupttor zu gehen, umrunden Sie das Gelände zur unteren linken (südwestlichen) Ecke. Sie werden eine kleine, verwitterte Holzhütte und mehrere Käfige sehen, die anscheinend für Vieh (Livestock) verwendet wurden – daher der Name der Mission.

Das Herantreten an diese Käfige oder das Betreten der Hütte löst normalerweise den Abschluss der ersten Teilaufgabe aus. Die Tür, die Sie jedoch tatsächlich öffnen müssen, ist mit einem digitalen Tastenfeld verschlossen, was einen Abstecher zu einem nahe gelegenen Wohngebäude erforderlich macht, um die Kombination zu finden.

## Schritt 2: Finden des Car Office Codes

Sie können die Bürotür nicht mit roher Gewalt öffnen; Sie müssen die physische Notiz mit dem Code finden. Von der Holzhütte und den Käfigen aus müssen Sie nach Norden gehen. Suchen Sie nach einem Wohnhaus, das sich direkt außerhalb des unmittelbaren Industriekreises befindet.

1. Verlassen Sie den Werkstattbereich und bewegen Sie sich nach Norden.
2. Betreten Sie das Haus (es hat normalerweise einen Fernseher im Hauptraum).
3. Suchen Sie in der Nähe des Fernsehgeräts nach einer kleinen Notiz.
4. Interagieren Sie mit der Notiz, um den Code zu enthüllen.

Obwohl der Code von einigen Operatoren als **2896** gemeldet wurde, können Codes in Gray Zone Warfare gelegentlich randomisiert oder in saisonalen Patches geändert werden. Interagieren Sie immer mit der Notiz, um sicherzustellen, dass das Ziel in Ihrem Aufgabenprotokoll aktualisiert wird.

## Schritt 3: Betreten des Büros und Abrufen der Intel

Kehren Sie mit dem Code in der Hand zum Bereich der Autowerkstatt zurück, in dem Sie zuerst die Käfige gefunden haben. Das Büro befindet sich innerhalb desselben Gebäudekomplexes, aber der Eingang ist oft auf der Rückseite versteckt. Suchen Sie nach einer Tür mit einem digitalen Vorhängeschloss.

Interagieren Sie mit dem Schloss und geben Sie den Code ein, den Sie im nördlichen Haus entdeckt haben. Sobald die Tür aufschwingt, betreten Sie das Büro und suchen Sie nach einem Schreibtisch oder einem Aktenschrank. Das Dokument mit der Aufschrift „Car Office Intel“ liegt dort bereit. Nehmen Sie es an sich und sichern Sie es sofort in Ihrem „Safe Container“, falls Sie Platz haben. Dies stellt sicher, dass Sie den Missionsgegenstand nicht verlieren, selbst wenn Sie während der Extraktion KIA (Killed in Action) gehen.

| Standort-Komponente | Beschreibung | Schlüsselinteraktion |
| :--- | :--- | :--- |
| **Nördliches Haus** | Wohngebäude mit Fernseher | **Notiz/Code finden** |
| **Südwestliche Hütte** | Holzstruktur bei Käfigen | **Initiales Ziel** |
| **Car Office** | Verschlossener Raum hinten im Shop | **Code eingeben & Intel looten** |

## Taktische Ratschläge für die Livestock-Aufgabe

Die **Grey Zone Warfare Livestock**-Mission findet in einem Gebiet mit weiten Sichtlinien und mehreren erhöhten Positionen statt, an denen KI-Scharfschützen spawnen können. Um diese Zone im Jahr 2026 erfolgreich zu navigieren, befolgen Sie diese taktischen Empfehlungen:

* **Schalldämpfer sind Pflicht:** Die Autowerkstatt ist eine „heiße“ Zone. Die Verwendung einer Waffe ohne Schalldämpfer wird jede KI im Umkreis von 200 Metern auf Ihre Position aufmerksam machen, was die Suche nach dem Code erheblich erschwert.
* **Überprüfen Sie die Dächer:** Bevor Sie sich dem nördlichen Haus nähern, suchen Sie die Dächer der umliegenden Industriegebäude mit dem Fernglas ab. Die KI sitzt oft an den Kanten mit SKS- oder AK-Gewehren.
* **Säubern Sie den Innenraum:** Wenn Sie das Haus für den Code betreten, stürmen Sie nicht hinein. Die KI kann manchmal im Inneren oder direkt hinter dem Haus spawnen. Benutzen Sie eine Blendgranate, wenn Sie einen „Closet Spawn“ vermuten.

> **💡 Tipp:** Wenn Sie in einem Trupp spielen, lassen Sie einen Spieler in der Nähe des Büros bleiben, während die anderen beiden den Code holen. Dies verhindert, dass die KI im Bürobereich respawnt, während Sie weg sind.

## Abschluss der Mission

Seien Sie vorsichtig, da LZs in der Nähe von Missionszentren häufige Ziele für „LZ-Camper“ oder umherwandernde KI-Patrouillen sind.

Zurück im Basislager öffnen Sie das Menü „Vendors“, wählen Banshee aus und navigieren zum Reiter „Tasks“. Klicken Sie auf „Livestock“ und wählen Sie „Hand Over“, um die Mission abzuschließen.

## FAQ

**F: Ist der Code für die Grey Zone Warfare Livestock-Aufgabe für jeden gleich?**

A: Während viele Spieler den Code 2896 melden, randomisiert Gray Zone Warfare Codes gelegentlich pro Server oder pro Spielerinstanz in neueren Updates. Es wird immer empfohlen, die Notiz im nördlichen Haus zu überprüfen, um sicherzustellen, dass der Code für Ihren spezifischen Missionsstatus aktiv ist.

**F: Was passiert, wenn ich sterbe, nachdem ich die Intel aufgehoben habe?**

A: Wenn sich die Intel in Ihrem Standard-Rucksack oder Rig befindet, lassen Sie sie bei Ihrem Tod fallen und müssen zu Ihrer Leiche zurückkehren, um sie zu bergen. Wenn Sie die Intel jedoch in Ihren Safe Container (Gamma/Epsilon/etc.) legen, bleibt sie bei Ihnen, auch wenn die Extraktion fehlschlägt.

**F: Benötige ich einen speziellen Schlüssel, um das Haus mit dem Code zu betreten?**

A: Nein, das Haus mit dem Code ist normalerweise unverschlossen. Sie benötigen nur den digitalen Code für die finale Bürotür, hinter der sich die Intel befindet.

**F: Kann ich diese Aufgabe in einem Trupp abschließen?**

A: Ja, aber es muss nur eine Person den Code finden, damit der ganze Trupp ihn benutzen kann. Jeder Spieler muss jedoch das Intel-Dokument physisch aufheben (oder warten, bis es respawnt), um seine individuellen Aufgabenanforderungen zu erfüllen, es sei denn, die Mission erlaubt einen geteilten Fortschritt.
