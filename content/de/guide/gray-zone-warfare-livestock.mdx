## Livestock Aufgaben-Übersicht

Bevor Sie ins Feld ziehen, ist es wichtig, die Ziele zu verstehen. Die Livestock-Aufgabe ist keine einfache "Suchen und Zerstören"-Mission. Sie beinhaltet Aufklärungs- und Rätselelemente, die tödlich sein können, wenn Sie nicht auf die Präsenz der lokalen Miliz vorbereitet sind.

| Ziel | Ort | Wichtige Voraussetzung |
| :--- | :--- | :--- |
| **Autowerkstatt untersuchen** | Spielberg (Nam Theun) | Käfige auf der Rückseite erreichen |
| **Büocode finden** | Wohnhaus (Norden) | Notiz in der Nähe des Fernsehers finden |

> **Warnung:** Das Gebiet Spielberg ist für eine hohe KI-Dichte bekannt. Erwarten Sie mehrere Gegner, die die Autowerkstatt und die umliegenden Wohnstraßen patrouillieren. Bringen Sie mindestens drei Ersatzmagazine und ausreichend medizinische Vorräte mit.

## Phase 1: Die Autowerkstatt finden

Ihr erstes Ziel ist das Gebiet Spielberg, speziell die verlassene Autowerkstatt in der Nähe der Region Nam Theun. Der effizienteste Weg, diesen Ort zu erreichen, ist der Flug zur **LZ November 1** oder **LZ November 2**. Von diesen Landezonen aus müssen Sie sich in Richtung des südlichen Stadtrands begeben.

Die Autowerkstatt ist ein markantes Industriegebäude. Um den ersten Teil der **Gray Zone Warfare Livestock** Aufgabe abzuschließen, müssen Sie nicht nur das Gebäude finden, sondern auch die Beweise für den Menschenhandel entdecken.

1.  Nähern Sie sich der Autowerkstatt von Süden oder Westen, um zu vermeiden, dass Sie direkt auf die Hauptstraße geraten.
2.  Suchen Sie nach einem kleinen Holzschuppen oder einem Außenbereich mit Eisenkäfigen.
3.  Sie müssen physisch an diese Käfige herantreten und sie berühren, um den Abschluss des Ziels "Untersuchen" auszulösen.
4.  Sobald die Unteraufgabe aktualisiert wurde, ist Ihr nächstes Ziel, den Zugangscode für das verschlossene Büro im Hauptgebäude zu finden.

## Phase 2: Den Code für das Werkstattbüro finden

Das Büro in der Autowerkstatt ist durch ein digitales Tastenfeld geschützt. Im Gegensatz zu einigen statischen Schlüsseln im Spiel wird der Code für die **Gray Zone Warfare Livestock** Aufgabe oft auf einer physischen Notiz an einem separaten Ort gefunden. Sie müssen ein Stück nördlich der Werkstatt in den Wohnsektor von Spielberg reisen.

Das spezifische Haus, das Sie suchen, befindet sich in der Nähe der Koordinaten **182 127**. Dieses Haus ist an einem Fernsehgerät im Hauptwohnbereich erkennbar.

*   **Schritt 1:** Gehen Sie von der Autowerkstatt aus entlang der Hauptstraße nach Norden.
*   **Schritt 2:** Suchen Sie das Haus bei 182 127.
*   **Schritt 3:** Betreten Sie das Gebäude und prüfen Sie den Tisch oder Ständer direkt vor dem Fernseher.
*   **Schritt 4:** Interagieren Sie mit der Notiz, um den Code zu lesen. 

> **Hinweis:** Der Code wird für verschiedene Spieler oder Sitzungen zufällig generiert. Häufig gemeldete Codes sind **2381** oder **2896**, aber Sie müssen die Notiz in Ihrer spezifischen Instanz überprüfen, um sicherzustellen, dass das Keypad die Eingabe akzeptiert.

## Phase 3: Die Dokumente bergen

Kehren Sie mit dem Code in der Hand zur Autowerkstatt zurück. Das Büro befindet sich im hinteren Teil der Hauptstruktur. Sie werden eine Tür sehen, die mit einem elektronischen Keypad ausgestattet ist.

| Aktion | Details |
| :--- | :--- |
| **Tür annähern** | Befindet sich an der Rückseite/Seite des Autowerkstatt-Gebäudes. |
| **Code eingeben** | Verwenden Sie den 4-stelligen Code aus dem Wohnhaus. |
| **Gegenstand finden** | Die Dokumente liegen normalerweise auf einem Schreibtisch oder Regal im Inneren. |
| **Gegenstand sichern** | Legen Sie die Dokumente sofort in Ihren **Safe Container**. |

Sobald Sie die Dokumente haben, erfordert die **Gray Zone Warfare Livestock** Aufgabe, dass Sie sicher extrahieren. Da Spielberg eine umkämpfte Zone ist, kann der Weg zurück zur LZ der gefährlichste Teil der Mission sein. Wenn Sie sterben, bevor Sie die LZ erreichen, verbleiben die Dokumente in Ihrem Safe Container, aber Sie müssen dennoch erfolgreich exfiltrieren, um sie Banshee zu übergeben.

## Taktische Empfehlungen für Spielberg

Die KI im Gebiet Nam Theun/Spielberg ist deutlich aggressiver als in den Startstädten. Sie tragen oft leichte Rüstung und führen Gewehre der AK-Serie mit sich, was sie selbst für Spieler im mittleren Spielverlauf zu einer Bedrohung macht.

1.  **LZ-Auswahl:** LZ November 2 bietet einen etwas verdeckteren Zugang durch die Waldgrenze, während November 1 Sie näher an die Hauptstraße bringt, wo Sie leicht entdeckt werden.
2.  **Schalldämpfer sind Pflicht:** Wenn Sie solo spielen, ist ein Schalldämpfer Ihr bester Freund. Unschallgedämpfte Schüsse locken die KI aus dem gesamten Block an Ihre Position.
3.  **Dächer prüfen:** Einige Gebäude in Spielberg haben Flachdächer oder Balkone, auf denen KI-Scharfschützen spawnen können. Scannen Sie immer vertikal, bevor Sie offene Straßen überqueren.
4.  **Team-Synergie:** Wenn Sie in einem Trupp spielen, sollte ein Spieler das Haus bei 182 127 sichern, während ein anderer den Code holt. Die Wohngebiete haben viele "L"-Ecken, in denen sich die KI verstecken kann.

## Fehlerbehebung bei häufigen Problemen

Viele Spieler stoßen während der **Gray Zone Warfare Livestock** Quest aufgrund der Art des "Untersuchen"-Triggers auf Schwierigkeiten. Wenn das Ziel nicht aktualisiert wird, versuchen Sie, um die Käfige herumzugehen oder sich hineinzustellen, falls die Türen offen sind.

Ein weiteres häufiges Problem ist, dass die Code-Notiz nicht erscheint. Wenn die Notiz im Haus bei 182 127 fehlt, stellen Sie sicher, dass Sie die Aufgabe von Banshee tatsächlich angenommen haben. In einigen Fällen kann ein Server-Neustart oder der Wechsel in eine andere Instanz erforderlich sein, falls die Weltgegenstände nicht korrekt spawnen.

| Komponente | Mögliches Problem | Lösung |
| :--- | :--- | :--- |
| **Käfige** | Ziel triggert nicht | Näher herangehen/neben den Käfigboden legen. |
| **Code-Notiz** | Notiz liegt nicht auf dem Tisch | Boden prüfen oder Sitzung neu starten. |
| **Keypad** | Code "Ungültig" | Notiz erneut lesen; Codes sind **einzigartig** pro Instanz. |
| **Extraktion** | LZ wird belagert | Zu einer weiter entfernten, sekundären LZ ausweichen. |

## FAQ

**F: Wo genau befindet sich die Autowerkstatt für die Gray Zone Warfare Livestock Aufgabe?**

A: Die Werkstatt befindet sich in der Stadt Spielberg, die Teil der Region Nam Theun ist. Es ist ein großes Industriegebäude auf der Südseite der Stadt, erkennbar an den Eisenkäfigen, die sich hinter oder neben dem Gebäude befinden.

**F: Ist der Büocode für jeden gleich?**

A: Nein, der Code für das Werkstattbüro ist zufällig. Während einige Spieler Codes wie 2381 oder 2896 gemeldet haben, müssen Sie die Notiz im Haus bei den Koordinaten 182 127 finden, um den korrekten Code für Ihre aktuelle Missionsinstanz zu erhalten.

**F: Was passiert, wenn ich die Livestock-Dokumente nach dem Aufheben verliere?**

A: Wenn Sie sterben und die Dokumente nicht in Ihrem Safe Container waren, müssen Sie zum Büro der Autowerkstatt zurückkehren, um sie erneut zu holen. Wenn Sie den Code bereits verwendet haben, kann die Tür entriegelt bleiben, oder Sie müssen den Code erneut eingeben.

**F: Welcher Händler vergibt die Livestock-Aufgabe?**

A: Die Livestock-Aufgabe wird von Banshee vergeben. Sie ist Teil der Questreihe des 0.4 Spearhead Updates und konzentriert sich darauf, die illegalen Aktivitäten lokaler krimineller Elemente in Lamang aufzudecken.
