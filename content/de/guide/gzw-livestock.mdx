Diese Mission erfordert eine Mischung aus Aufklärung, Erkundung und taktischer Extraktion, während Sie durch ein feindseliges Dorf navigieren, um dunkle Geheimnisse über lokale Aktivitäten aufzudecken. ## GZW Livestock: Aufgabenübersicht und Anforderungen

Bevor Sie ins Feld ziehen, ist es wichtig zu verstehen, was Banshee von Ihnen erwartet. Bei dieser Aufgabe geht es darum, Berichten über illegalen "Viehtransport" (Livestock) nachzugehen, was sich als weitaus finsterer herausstellt als einfache Tierhaltung. Sie werden in einem stark frequentierten Gebiet operieren, daher ist die richtige Ausrüstung nicht verhandelbar.

| Anforderung | Detail |
| :--- | :--- |
| **Händler** | **Banshee** |
| **Ort** | Südwestlich der Autowerkstatt / Nördliches Haus |
| **Ziele** | Käfige lokalisieren, Büro-Code finden, Informationen bergen |
| **Empfohlenes Level** | 15+ (Spearhead Update) |
| **Benötigter Gegenstand** | Randomisierter Code (in der Mission zu finden) |

> **💡 Tipp:** Stellen Sie sicher, dass Sie mindestens zwei leere Inventarplätze (2x1) für das Dokument haben, das Sie am Ende der Mission bergen müssen.

## Schritt 1: Lokalisieren der Käfige und der Holzhütte

Die erste Phase der **gzw livestock**-Mission beginnt am südwestlichen Rand des markierten Zielkreises. Sie suchen nach einer bestimmten Holzhütte und einer Reihe von Käfigen, die darauf hindeuten, wo das "Vieh" gefangen gehalten wurde.

1.  **Anfahrt zur Autowerkstatt**: Begeben Sie sich in Richtung der Industriezone im Dorf.
2.  **Nach Südwesten navigieren**: Bewegen Sie sich in die untere linke Ecke des Werkstattgeländes.
3.  **Die Hütte identifizieren**: Suchen Sie nach einem kleinen, verwitterten Holzgebäude, das von Käfigen umgeben ist.
4.  **Ziel triggern**: Das bloße Herantreten an diese Käfige sollte den Abschluss des ersten Teilziels in Ihrem Menü auslösen.

Dieses Gebiet wird oft von 2-3 KI-Kämpfern patrouilliert. Es wird empfohlen, den Perimeter zu säubern, bevor man offen bei den Käfigen stehen bleibt, da die Hütte kaum harten Schutz vor eintreffendem Feuer bietet.

## Schritt 2: Den Code für das Büro finden

Sobald Sie den Standort der Käfige bestätigt haben, werden Sie feststellen, dass das Hauptbürogebäude in der Nähe mit einem Tastenfeld oder Vorhängeschloss gesichert ist, das einen spezifischen Code erfordert. Im Gegensatz zu einigen Schlüsseln in Gray Zone Warfare, die von jeder KI erbeutet werden können, befindet sich der **gzw livestock**-Code in einem festen Wohngebäude im Norden.

| Ziel | Standortbeschreibung | Gegenstand |
| :--- | :--- | :--- |
| **Haus durchsuchen** | Nördlich des Missionskreises | **Notiz mit Code** |
| **Raum identifizieren** | Wohnbereich in der Nähe des Fernsehers | **Büro-Code** |
| **Code-Variation** | Einzigartig pro Instanz (z. B. 2896) | **Zugang zu Infos** |

Gehen Sie von den Käfigen aus nach Norden zu den Wohnhäusern. Suchen Sie nach einem Haus, das im Vergleich zu den umliegenden Ruinen relativ intakt erscheint. Betreten Sie das Gebäude und suchen Sie das Fernsehgerät. Auf einem kleinen Tisch oder Ständer in der Nähe des Fernsehers finden Sie eine Notiz. Die Interaktion mit dieser Notiz offenbart Ihren spezifischen Code.

> **Warnung:** Der Code wird für verschiedene Spieler oder Instanzen randomisiert. Verlassen Sie sich nicht auf Codes aus alten Forenbeiträgen; Sie müssen physisch mit der Notiz in Ihrer aktuellen Sitzung interagieren, um fortzufahren.

## Schritt 3: Das Büro betreten und Informationen extrahieren

Kehren Sie mit dem Code in der Hand zum ursprünglichen Zielort bei den Käfigen zurück. Der Büroeingang befindet sich normalerweise an der Rückseite des Gebäudes neben der Holzhütte.

Befolgen Sie diese Schritte, um die **gzw livestock**-Aufgabe abzuschließen:
*   **Vorhängeschloss finden**: Gehen Sie zur Hintertür des Bürogebäudes.
*   **Code eingeben**: Verwenden Sie den Code, den Sie im nördlichen Haus gefunden haben, um die Tür zu entriegeln.
*   **Dokumente greifen**: Die Informationen liegen normalerweise auf einem Schreibtisch oder einem Aktenschrank im kleinen Büroraum.
*   **Gegenstand sichern**: Legen Sie das Dokument in Ihren Sicherheitscontainer (falls es passt) oder Ihren Rucksack.

Sobald Sie das Dokument besitzen, hat die Extraktion Priorität. Das Dorf ist eine berüchtigte "Hot Zone" für Spieler-Begegnungen (PvP), besonders in der Nähe von missionskritischen Gebäuden.

## Häufige Gefahren und Tipps

Das Gebiet um die **gzw livestock**-Ziele ist trügerisch gefährlich. Während die KI spärlich gesät scheint, nutzen sie oft das dichte Laub und baufällige Mauern für Hinterhalte.

*   **Dächer prüfen**: Die Autowerkstatt hat mehrere erhöhte Positionen, an denen KI mit SKS-Gewehren spawnen kann. Scannen Sie immer die Skyline, bevor Sie sich den Käfigen nähern.
*   **Der "unsichtbare" Code**: Wenn die Notiz nicht in der Nähe des Fernsehers erscheint, versuchen Sie sich neu einzuloggen oder prüfen Sie die Küchenzeile. In seltenen Fällen kann das Objekt durch die Möbel glitchen.
*   **Trupp-Koordination**: Wenn Sie in einer Gruppe spielen, lassen Sie einen Spieler am Büro bleiben, während die anderen zwei den Code holen. Dies verhindert, dass das Gebiet von feindlichen Patrouillen neu besetzt wird, während Sie weg sind.
*   **Extraktionsweg**: Nehmen Sie nicht die Hauptstraße zurück zur Landezone (LZ). Nutzen Sie die Waldlinie westlich der Autowerkstatt, um nicht von Trupps entdeckt zu werden, die gerade erst ihre eigenen Missionen starten.

## Missionsbelohnungen

Nach erfolgreicher Extraktion und Übergabe der Informationen an Banshee erhalten Sie einen signifikanten Ruf-Boost. Dies ist essenziell, um höherwertige Schalldämpfer und spezialisierte Munition freizuschalten.

*   **Erfahrungspunkte**: 1.200 XP
*   **Händler-Ruf**: +150 (Banshee)
*   **Währung**: ca. 4.500 $
*   **Ausrüstung**: Enthält meist seltene medizinische Vorräte oder Munitionskisten.

## FAQ

**F: Kann ich den gzw livestock Code finden, bevor ich die Käfige besuche?**

A: Ja, Sie können technisch gesehen direkt zum Haus gehen, um den Code zu holen, aber das Missionslog aktualisiert sich möglicherweise erst korrekt, wenn Sie zuerst die südwestlichen Käfige besucht haben. Es ist sicherer, die Ziele in der Reihenfolge abzuarbeiten.

**F: Ändert sich der Code, wenn ich sterbe und zurückkomme?**

A: Nein, der Code ist normalerweise an die spezifische Missionsinstanz Ihres Charakters gebunden. Wenn Sie jedoch einem anderen Server oder dem Trupp eines Freundes beitreten, sollten Sie die Notiz erneut prüfen, um sicherzustellen, dass sich der Code nicht geändert hat.

**F: Wird der Büro-Schlüssel/Code für andere Missionen verwendet?**

A: Derzeit ist der in der **gzw livestock**-Aufgabe gefundene Code spezifisch für dieses Ziel. Das Büro selbst enthält jedoch kleinere Loot-Spawns, die für spätere Raids nützlich sein können.

**F: Was passiert, wenn ich das Dokument verliere?**

A: Wenn Sie sterben, bevor Sie extrahieren oder das Dokument in Ihren Sicherheitscontainer legen, müssen Sie zum Büro zurückkehren. Wenn die Tür bereits entriegelt ist, können Sie einfach hineingehen und es erneut holen. Wenn die Tür zurückgesetzt wurde, müssen Sie den Code möglicherweise erneut am nördlichen Haus verifizieren.
