Das 0.4 Spearhead-Update für Gray Zone Warfare hat eine Reihe herausfordernder neuer Aufgaben eingeführt, die Spieler in die gefährlicheren Gebiete von Lamang führen. Darunter befindet sich die **gray zone livestock** Aufgabe, eine Mission von Banshee, die taktische Präzision und scharfe Beobachtungsgabe erfordert. Diese Aufgabe sendet Operator in das Gebiet Spielberg, um düsteren Berichten über Menschenhandel nachzugehen, die sich um eine verlassene Autowerkstatt konzentrieren. Das Abschließen der **gray zone livestock** Mission ist essenziell für Spieler, die ihren Ruf bei Banshee verbessern und höherwertige Ausrüstung in der Meta von 2026 freischalten möchten.

Das Navigieren in den Außenbezirken von Nam Theun kann tückisch sein, besonders mit der erhöhten KI-Dichte, die im neuesten Patch eingeführt wurde. Dieser Guide bietet eine schrittweise Aufschlüsselung der Ziele, einschließlich der exakten Koordinaten für die Beweise, den Ort des randomisierten Büro-Codes und die besten Extraktionsrouten, um sicherzustellen, dass Sie mit den Dokumenten in der Hand zur Basis zurückkehren.

## Aufgabenübersicht und Anforderungen

Bevor Sie ins Feld ziehen, ist es wichtig, den Umfang der Mission zu verstehen. Banshee sucht gezielt nach handfesten Beweisen für illegale Aktivitäten. Sie werden nicht nur Feinde ausschalten; Sie agieren als Ermittler in einer Umgebung mit hoher Bedrohung.

| Aufgabenelement | Details |
| :--- | :--- |
| **Händler** | **Banshee** |
| **Standort** | **Spielberg (Nam Theun Gebiet)** |
| **Primärziel** | Autowerkstatt nach Beweisen für Menschenhandel untersuchen |
| **Sekundärziel** | Büro-Code finden und Dokumente bergen |
| **Empfohlene LZ** | **November 1 oder November 2** |
| **Gefahrenstufe** | **Mittel-Hoch** |

> **Warnung:** Das Spielberg-Gebiet ist berüchtigt für KI-Scharfschützen, die im umliegenden Gebüsch versteckt sind. Stellen Sie sicher, dass Ihr Helm eine hohe ballistische Schutzklasse hat, und führen Sie mindestens zwei chirurgische Kits mit sich.

## Schritt 1: Die Autowerkstatt finden

Die erste Phase der **gray zone livestock** Aufgabe erfordert, dass Sie in die Region Spielberg fliegen. Die meisten Spieler finden, dass die Landung bei **LZ November 2** den direktesten Zugang bietet, während **November 1** eine heimlichere Flankenroute durch den Wald ermöglicht.

Sobald Sie ankommen, begeben Sie sich zur verlassenen Autowerkstatt. Dieses Gebäude dient als zentraler Knotenpunkt für die Mission. Ihr erstes Ziel ist es, physische Beweise für Menschenhandel zu finden. Um dieses Ziel abzuschließen, müssen Sie die Außenkäfige lokalisieren.

### Die Beweise finden
Die Käfige befinden sich in einem kleinen Holzschuppen/Struktur an der unteren linken Ecke (Südwesten) des Hauptgebäudes der Autowerkstatt. Sie müssen nicht unbedingt mit einem bestimmten Gegenstand interagieren; normalerweise wird der Abschluss des Teilziels ausgelöst, wenn Sie sich den Metallstäben der Käfige nähern oder diese berühren.

| Zielort | Koordinaten / Beschreibung |
| :--- | :--- |
| **Menschenhandel-Käfige** | Südwestliche Ecke des Werkstattgeländes |
| **Hauptbüro** | Befindet sich an der Rückseite des Werkstattgebäudes |
| **Code-Haus** | Nördlich der Werkstatt bei **182 127** |

## Schritt 2: Den Code für das Büro finden

Nachdem Sie die Anwesenheit der Käfige bestätigt haben, aktualisiert sich die **gray zone livestock** Aufgabe und verlangt von Ihnen, das verschlossene Büro an der Rückseite der Werkstatt zu betreten. Diese Tür ist mit einem digitalen Vorhängeschloss gesichert, und Sie können sie nicht mit Sprengstoff oder Dietrichen öffnen. Sie müssen den spezifischen Code finden, den die Betreiber hinterlassen haben.

Verschwenden Sie keine Zeit damit, in der Werkstatt nach dem Code zu suchen. Gehen Sie stattdessen von der Werkstatt aus nach Norden in Richtung der Wohnsiedlung. Sie suchen ein bestimmtes Haus, das sich etwa bei den Koordinaten **182 127** befindet.

1.  Bewegen Sie sich von der Autowerkstatt nach Norden.
2.  Betreten Sie das Haus direkt vor dem Fernsehgerät.
3.  Suchen Sie nach einer kleinen Notiz oder einem Zettel, der auf einem Tisch oder in der Nähe des Fernsehers liegt.
4.  **Hinweis:** Dieser Code ist für jeden Spieler randomisiert. Häufige Codes, die im Feld gesehen wurden, sind **2896** oder **2381**, aber Sie müssen die Notiz selbst prüfen, um sicherzustellen, dass Sie den richtigen Code für Ihre Sitzung haben.

## Schritt 3: Die Dokumente beschaffen

Kehren Sie mit dem eingeprägten (oder aufgeschriebenen) Code zur Autowerkstatt zurück. Gehen Sie zur Rückseite des Gebäudes, wo Sie eine Tür mit einem Tastenfeld finden.

### Das Büro betreten
Interagieren Sie mit dem Schloss und geben Sie den 4-stelligen Code ein, den Sie im nördlichen Haus gefunden haben. Im Inneren ist das Büro relativ klein. Suchen Sie nach einem Stapel Papier oder einem Ordner – dies sind die Beweise, die Banshee benötigt.

> **💡 Tipp:** Sobald Sie die Dokumente aufheben, belegen sie einen Platz in Ihrem Quest-Inventar. Wenn Sie vor der Extraktion getötet werden, könnten Sie den Gegenstand verlieren, abhängig von den Persistenzregeln Ihrer Fraktion im 2026-Build. Es wird dringend empfohlen, sich direkt zur LZ zu begeben, sobald Sie die Papiere haben.

| Zu bergender Gegenstand | Visuelle Beschreibung |
| :--- | :--- |
| **Handelsbuch** | Ein gelber/beiger Ordner auf dem Schreibtisch |
| **Personalunterlagen** | Oft in der Nähe des Aktenschranks zu finden |
| **Das Schlüsseldokument** | Speziell für die "Livestock"-Aufgabe beschriftet |

## Taktische Ratschläge für Spielberg

Die **gray zone livestock** Mission findet in einem Gebiet mit langen Sichtlinien und viel Deckung für die KI statt. Spielberg ist kein Ort, an dem Sie sich auf ein langwieriges Feuergefecht einlassen sollten.

### Empfohlene Ausrüstung
*   **Waffe:** Ein Sturmgewehr mit einer 4x oder 6x Optik ist ideal. Die Distanzen zwischen dem Code-Haus und der Werkstatt sind gerade lang genug, um Rotpunktvisiere unzureichend erscheinen zu lassen.
*   **Munition:** Verwenden Sie panzerbrechende Munition (AP). Die lokale Miliz im 0.4 Update trägt oft Level II oder III Weichballistik.
*   **Hilfsmittel:** Rauchgranaten sind unglaublich nützlich, um die Straße zwischen der Werkstatt und den nördlichen Häusern zu überqueren, da die KI Spieler oft über den offenen Asphalt verfolgt.

### Extraktionsstrategie
Der nächste Extraktionspunkt ist normalerweise der, an dem Sie angekommen sind (November 2). Wenn Sie jedoch heftiges Gewehrfeuer hören oder hohe Spieleraktivität bei N2 sehen, ziehen Sie in Erwägung, nach Osten zum Fluss zu wandern und einen Bogen zu einer sekundären LZ zu machen. Die Dokumente haben Priorität; vermeiden Sie unnötige Kämpfe, sobald das Ziel abgeschlossen ist.

## Zusammenfassung der Schritte

Um einen erfolgreichen Durchlauf der **gray zone livestock** Aufgabe zu gewährleisten, folgen Sie dieser kompakten Checkliste:

1.  Nehmen Sie die Aufgabe von **Banshee** im Basislager an.
2.  Infiltrieren Sie bei **LZ November 2**.
3.  Begeben Sie sich zur **Autowerkstatt** und gehen Sie zu den Käfigen im Holzschuppen.
4.  Reisen Sie nach Norden zum Haus bei **182 127** und lesen Sie den Code beim Fernseher.
5.  Kehren Sie zur Werkstatt zurück und geben Sie den Code in das Büroschloss ein.
6.  Nehmen Sie die Dokumente vom Schreibtisch.
7.  Extrahieren Sie sicher und übergeben Sie die Dokumente an Banshee.

## FAQ

**F: Ist der Büro-Code für jeden in der Gray Zone Livestock Aufgabe gleich?**

A: Nein. Der Code ist randomisiert. Während einige Spieler Codes wie 2896 oder 2381 berichten, müssen Sie das Haus bei den Koordinaten 182 127 besuchen, um den spezifischen Code zu finden, der für Ihre Sitzung generiert wurde.

**F: Ich habe die Käfige erreicht, aber die Aufgabe wurde nicht aktualisiert. Was soll ich tun?**

A: Der Triggerbereich für die Käfige kann schmal sein. Versuchen Sie, in den kleinen Schuppen zu gehen und direkt an den Käfigtüren zu stehen. Stellen Sie sicher, dass Sie sich bei der Holzstruktur südwestlich des Hauptwerkstattgebäudes befinden.

**F: Kann ich diese Aufgabe in einem Trupp abschließen?**

A: Ja, aber jedes Mitglied des Trupps muss seinen eigenen Satz Dokumente aus dem Büro aufheben. Der Code sollte jedoch für alle Mitglieder desselben Trupps in dieser Instanz identisch sein.

**F: Was passiert, wenn ich sterbe, nachdem ich den Code gefunden habe, aber bevor ich die Dokumente habe?**

A: Normalerweise müssen Sie den Code nicht erneut suchen, wenn Sie ihn sich gemerkt oder aufgeschrieben haben. Sie können bei Ihrem nächsten Einsatz direkt zum Büroschloss gehen. Wenn Sie jedoch sterben, nachdem Sie die Dokumente aufgehoben haben, müssen Sie diese wahrscheinlich erneut bergen.
