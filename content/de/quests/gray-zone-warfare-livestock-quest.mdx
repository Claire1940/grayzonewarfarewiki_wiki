Eine der komplexeren Aufgaben, die mit dem 0.4 Spearhead-Update eingeführt wurden, ist die **Gray Zone Warfare Livestock Quest** – eine Mission, bei der Spieler in ein Hochrisikogebiet eindringen müssen, um sensible Dokumente zu bergen. Diese Mission ist bekannt für ihren mehrstufigen Prozess: Spieler müssen eine bestimmte Holzhütte auskundschaften, einen versteckten Zugangscode finden und schließlich in ein verschlossenes Büro eindringen. Der erfolgreiche Abschluss der **Gray Zone Warfare Livestock Quest** ist entscheidend für Spieler, die ihren Ruf bei Händlern verbessern und hochwertigere taktische Ausrüstung freischalten möchten.

## Quest-Übersicht und Anforderungen

Bevor Sie ins Feld ziehen, ist es wichtig, die Parameter der Mission zu verstehen. Die Livestock-Aufgabe wird in der Regel von Turncoat oder Handshake zugewiesen, abhängig von Ihrem Fraktionsfortschritt, und findet in der Nähe eines schwer bewachten Industriesektors statt.

| Merkmal | Details |
| :--- | :--- |
| **Quest-Name** | Livestock |
| **Standort** | Autowerkstatt / Nördliche Industriezone |
| **Hauptziel** | Dokumente aus dem Autobüro bergen |
| **Benötigter Gegenstand** | Autobüro-Code (im Einsatz zu finden) |
| **Schwierigkeit** | Moderat |
| **Geschätzte Zeit** | 15–25 Minuten |

> **Achtung:** Das Gebiet um die Autowerkstatt wird oft von KI mit Präzisionsgewehren patrouilliert. Stellen Sie sicher, dass Sie mindestens eine Körperpanzerung der Klasse 3 tragen, bevor Sie dies solo versuchen.

## Phase 1: Die Holzhütte lokalisieren

Der erste Schritt der **Gray Zone Warfare Livestock Quest** besteht darin, eine bestimmte Holzhütte auszukundschaften. Dieses Gebäude dient als erster Wegpunkt und befindet sich in der Nähe der Autowerkstatt.

Um sie zu finden, nähern Sie sich der Autowerkstatt von Südwesten. Suchen Sie nach einem kleinen, verwitterten Holzgebäude an der unteren linken Ecke des Werkstattgeländes. Sie müssen in dieser ersten Phase nicht unbedingt einen bestimmten Gegenstand in dieser Hütte finden; das bloße Betreten oder Stehen in unmittelbarer Nähe löst normalerweise die Benachrichtigung „Ziel abgeschlossen“ für den Erkundungsteil der Aufgabe aus.

### Checkliste zur Erkundung
1.  **Von Süden nähern:** Meiden Sie die Hauptstraße, um die Sichtbarkeit zu minimieren.
2.  **Hütte identifizieren:** Es handelt sich um ein kleines, freistehendes Holzgebäude.
3.  **Umgebung sichern:** Überprüfen Sie das hohe Gras in der Nähe auf liegende KI-Kämpfer.
4.  **Ziel bestätigen:** Warten Sie auf die UI-Benachrichtigung, bevor Sie zur nächsten Phase übergehen.

## Phase 2: Den Code für das Autobüro finden

Sobald die Hütte erkundet wurde, haben Sie noch keinen direkten Zugriff auf die verschlossenen Dokumente. Sie müssen zuerst den Keypad-Code finden. Viele Spieler machen den Fehler, das Schloss mit Gewalt öffnen zu wollen oder in der Werkstatt selbst zu suchen, aber der Code befindet sich tatsächlich weiter nördlich.

Begeben Sie sich zum nördlichen Rand des Missionskreises. Suchen Sie nach einem kleinen Büro oder Sicherheitsposten. Im Inneren finden Sie eine Notiz oder ein Terminal mit dem **Autobüro-Code**. Während einige Codes in Gray Zone Warfare statisch sind, können andere je nach Server-Instanz oder Update variieren. Basierend auf aktuellen Informationen wird häufig der Code **2896** verwendet, aber Sie sollten ihn immer am nördlichen Standort verifizieren, um sicherzustellen, dass das Quest-Log korrekt aktualisiert wird.

| Fundort des Codes | Beschreibung | Optischer Hinweis |
| :--- | :--- | :--- |
| **Nördlicher Sicherheitsposten** | Kleiner Schreibtisch im nördlichsten Gebäude | Suchen Sie nach einem gelben Post-it oder Ordner |
| **Alternativer Ort** | Wandmontiertes Klemmbrett | In der Nähe des Eingangs der Sicherheitshütte |

## Phase 3: Die Dokumente bergen

Kehren Sie mit dem Code in der Hand zum ursprünglichen Zielgebiet bei der Holzhütte zurück. Die Dokumente befinden sich in einem verstärkten Bereich des Gebäudes, der oft als „Autobüro“ bezeichnet wird.

Der Eingang zu diesem Büro befindet sich normalerweise an der Rückseite des Gebäudes. Suchen Sie nach einer Tür, die mit einem digitalen Vorhängeschloss gesichert ist. Interagieren Sie mit dem Keypad und geben Sie den Code ein, den Sie in Phase 2 erhalten haben. Sobald Sie drinnen sind, finden Sie die Dokumente meist auf einem Schreibtisch oder in einem Aktenschrank. 

> **💡 Tipp:** Sobald Sie die Dokumente aufheben, belegen sie einen Platz in Ihrem Quest-Inventar. Wenn Sie vor der Extraktion sterben, müssen Sie den Bergungsprozess möglicherweise neu starten, obwohl einige Checkpoints aktiv bleiben könnten.

## Empfohlene Ausrüstung für die Livestock Quest

Die industrielle Beschaffenheit der Autowerkstatt bedeutet, dass Sie mit einer Mischung aus Fernkämpfen und engen Raum-zu-Raum-Säuberungen konfrontiert werden. Die Wahl der richtigen Ausrüstung ist für die **Gray Zone Warfare Livestock Quest** entscheidend.

| Ausrüstungstyp | Empfehlung | Grund |
| :--- | :--- | :--- |
| **Waffe** | M4A1 oder AK-MN | Vielseitigkeit auf mittlere Distanz und hohe Mannstoppwirkung. |

| **Munition** | AP (Panzerbrechend) | KI in dieser Region trägt oft weiche Körperpanzerung. |
| **Medizinisch** | 2x Tourniquets, 1x Chirurgie-Kit | Hohe Wahrscheinlichkeit von Gliedmaßenschäden beim Eindringen. |

## Taktische Ratschläge für Solo-Spieler

Das Abschließen der **Gray Zone Warfare Livestock Quest** als Einzelkämpfer erfordert ein heimlicheres Vorgehen als im Trupp. Die KI im 0.4 Spearhead-Update wurde so angepasst, dass sie stärker auf Geräusche und Mündungsfeuer reagiert.

1.  **Schalldämpfer verwenden:** Dies ist für Solo-Spieler unverzichtbar. Die Reduzierung Ihrer akustischen Signatur ermöglicht es Ihnen, Wachen um die Holzhütte auszuschalten, ohne die gesamte Werkstatt zu alarmieren.
2.  **Vorteil der erhöhten Position:** Bevor Sie das Hüttengebiet betreten, kundschaften Sie von den Hügeln im Westen aus. Nutzen Sie Ihren Entfernungsmesser, um Gegnerpositionen zu markieren.
3.  **Geduld ist der Schlüssel:** Wenn die Autowerkstatt „heiß“ ist (viel Schusswechsel), warten Sie, bis andere Spieler das Gebiet geräumt haben oder bis die KI ihre Patrouillenwege zurückgesetzt hat.
4.  **Extraktionswege:** Nehmen Sie nicht den direktesten Weg zur Landezone (LZ). Gehen Sie durch den Wald zurück, um Campern auszuweichen, die möglicherweise die Hauptstraßen beobachten.

## FAQ

**Q: Wo genau befindet sich die Holzhütte für die Gray Zone Warfare Livestock Quest?**

A: Die Holzhütte befindet sich an der unteren linken (südwestlichen) Ecke des Autowerkstatt-Areals. Es ist ein kleines, freistehendes Gebäude, das als erster Zielpunkt für die Mission dient.

**Q: Muss ich den Code jedes Mal finden, wenn ich die Mission versuche?**

A: Wenn Sie den Code bereits gefunden haben und er in Ihren Quest-Notizen vermerkt ist, können Sie normalerweise direkt zum Keypad gehen. Wenn jedoch die Server-Instanz zurückgesetzt wird oder Ihr Quest-Fortschritt nicht getrackt wird, müssen Sie den nördlichen Sicherheitsposten möglicherweise erneut aufsuchen.

**Q: Was soll ich tun, wenn sich die Tür des Autobüros nicht öffnen lässt?**

A: Stellen Sie sicher, dass Sie den korrekten Code vom nördlichen Standort haben. In einigen Fällen versuchen Spieler, Codes aus älteren Versionen des Spiels zu verwenden. Überprüfen Sie im 0.4-Update immer die Notiz im nördlichen Büro, um die Ziffern zu bestätigen.

**Q: Kann ich den Code mit meinen Teamkameraden teilen?**

A: Ja, wenn Sie in einem Trupp sind, ermöglicht das Finden des Codes durch eine Person normalerweise dem gesamten Team den Fortschritt, vorausgesetzt, alle befinden sich in derselben Phase der **Gray Zone Warfare Livestock Quest**. Jeder Spieler muss jedoch die Dokumente physisch aufheben, um seine individuelle Aufgabe abzuschließen.
