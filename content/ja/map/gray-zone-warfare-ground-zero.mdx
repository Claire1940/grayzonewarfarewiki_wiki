<Section icon={Shield} title="サバイバル必需品：ラマング島での生き残り">
Gray Zone Warfareは抽出シューターであり、すべてのレイドには装備を失うリスクが伴います。特にグラウンドゼロで開始する際は、基本的なサバイバルメカニクスを理解し管理することが重要です。予防可能な死亡を避けるために、体力、水分補給、エネルギーレベルに細心の注意を払ってください。

<CardGrid cols={2}>
  <Card title="水分補給とエネルギー">
    - **喉の渇き:** 水分補給が50%未満になると、血液再生が停止します。
    - **脱水症状:** 水分補給が1%になると、視界がかすみます。
    - **空腹:** エネルギーが50%未満になると、血液再生が停止します。
    - **飢餓:** エネルギーが1%になると、視界がかすみます。
    - **常に水と食料を携帯してください。**
  </Card>
  <Card title="負傷と昏睡">
    - **出血:** 軽度、中度、重度の出血は継続的な失血を引き起こします。すぐに治療してください。
    - **昏睡:** 血液が50%未満になった場合、または臓器（肺/肝臓）が破壊された場合に発生する可能性があります。
    - **蘇生可能時間:** 分隊員は昏睡状態のプレイヤーを約180秒以内に蘇生できます。
    - **IFAKや包帯などの医療品**は必須です。
  </Card>
</CardGrid>

セキュアロックボックスは、グラウンドゼロでのあなたの親友です。中に置かれたアイテムは、死亡時に失われることから保護されます。これは、クエストアイテムや失うことのできない高価値の戦利品にとって特に重要です。

<Callout type="warning" title="貴重品を確保する">
常にクエストアイテムと保持したい高価値の戦利品は、セキュアロックボックスに入れてください。ミッションに不可欠なアイテムを失うと、進行が大幅に妨げられ、タスクをやり直すことになります。
</Callout>

<Section icon={Sparkles} title="グラウンドゼロでの戦利品と経済の最適化">
Gray Zone Warfareの初期段階で安定した経済を確立することは、継続的な装備の調達とタスクの完了のために不可欠です。グラウンドゼロでは、低リスクの戦利品ルートと賢い販売方法に焦点を当ててください。タスク報酬は基本的な収入を提供しますが、効率的な略奪は資金を大幅に増やします。

<Callout type="info" title="お金と略奪の哲学">
Gray Zone Warfareにおけるお金は、タスク報酬と賢い略奪という2つの安定した源から生まれます。初期のタスクは現金と評判を提供し、FOBのベンダーネットワークは貴重品を販売することを可能にします。良い回復ループには、近くのタスクを完了し、効率的に略奪し、賢く販売し、報酬を使って高価なキットに過度に費やすことなく再補給することが含まれます。
</Callout>

### 効率的な略奪と販売

| 行動                               | 優先度 | 詳細                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| :----------------------------------- | :------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **有料タスクの完了**              | 高     | ハンドシェイク、グニー、ラボラットからの多くの初期タスクは、経験値と評判に加えて直接的な現金報酬を提供します。例として、「医療品が欲しい」（1,200ドル）、「助けの手」（440ドル）、「店にあるもの」（3,300ドル）などがあります。これらは信頼できる収入源です。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **銃器コンテナの優先**    | 高     | 略奪時には、銃器コンテナ（CSI銃器ボックス、UN銃器ボックス、銃器ボックス）に焦点を当ててください。これらには、通常、ランダムなガラクタよりも価値のある武器、マガジン、武器部品が含まれています。ミリタリー銃器クレートからは、**AK308 7.62x51mm 20連マガジン**や**AK74 5.45x39mm 60連マガジン**のような高ティアのマガジンが得られることもあります。                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **最も高値で買い取るベンダーに売却**       | 中     | 貴重品はどのベンダーにも売却できますが、特定のアイテムに対してより高値で買い取るベンダーもいます。ゲームのWikiやコミュニティリソースを参照して、各アイテムの最も高値で買い取る買い手を見つけ、収益を最大化してください。専門外のベンダーに売却すると、通常50%の価格割引が適用されます。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **現金をベンダーアクセスに変換**     | 高     | 獲得した通貨を使って、タスク完了に役立つアイテムを購入し、それによってベンダーの評判を高めます。ベンダーの評判が高まると、より良い在庫がアンロックされ、優れた武器、アーマー、医療品を購入できるようになります。これにより、進行のためのポジティブなフィードバックループが生まれます。アップデート0.4では、報酬がアイテムよりも現金に重点を置くように再調整され、現金管理がさらに重要になりました。                                                                                                                                                                                                                                                                                                                                                                                                    |
| **低リスクの略奪ルート**             | 高     | 初期レイドでは、敵対的なNPCやプレイヤーの交通量が少ないエリアに留まってください。スタータータウンとその周辺には、高リスクの戦闘に従事することなく、十分な低ティアの戦利品があり、十分な貯蓄を築くことができます。初期タスクに関連するコンテナや特定の略奪場所に焦点を当ててください。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **セキュアロックボックスの使用**             | 致命的 | 常に保持したいアイテムにはセキュアロックボックスを使用してください。これには、高価値のコンポーネント、クエストアイテム、または珍しい発見が含まれます。これにより、レイド中に排除された場合に重要なリソースを失うことを防ぎます。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |

</Section>
