## Gray Zone Warfareにおける脱出の仕組み

Gray Zone Warfareの脱出システムは、プレイヤーを各勢力のベースキャンプとマップ各地に点在するランディングゾーン（LZ）の間で運ぶヘリコプター（リトルバード）を中心に構築されています。これらのLZは単にマップを去るための場所ではなく、主要な挿入（出撃）手段でもあります。

脱出を開始するには、タクティカルマップを操作する必要があります。マップ（デフォルトキー：Tab）を開くと、発見済みのすべての**Gray Zone Warfareの脱出ポイント**を確認できます。重要なのは、今立っているLZをクリックして去るのではないということです。代わりに、目的地（通常は自勢力のベースキャンプ）を選択してから、輸送を要請する必要があります。

| ステップ | アクション | 説明 |
| :--- | :--- | :--- |
| **1** | **マップを開く** | Tabキーを押して現在地と近くのLZを確認します。 |
| **2** | **目的地を選択** | ベースキャンプ、または移動したい別のLZをクリックします。 |
| **3** | **輸送を要請** | UIの「Call Transport」ボタンをクリックします。 |
| **4** | **到着を待つ** | タイマーとマップ上のヘリコプターのアイコンを監視します。 |
| **5** | **搭乗** | ヘリが着陸したら近づき、インタラクトキーを長押しして搭乗します。 |

> **💡 ヒント:** ヘリを呼ぶ前に必ず周囲を確認してください。リトルバードの接近音は、LZ付近に潜んでいる敵AIやライバルプレイヤーにとっての「ディナーベル（呼び出し鈴）」として機能します。

## ランディングゾーンの管理と待ち時間

**Gray Zone Warfareの脱出ポイント**を利用する上で最も複雑な側面の一つが、ヘリコプターの可用性に関するロジスティクスです。ゲームの世界では、同時に空中に存在できるアクティブなヘリコプターの数に厳格な制限があります。現在、1勢力あたり最大4機までとなっています。輸送を要請して「Unavailable（利用不可）」というメッセージが表示された場合、すべての輸送資産が現在使用中であることを意味します。

### 4機制限への対策
サーバーが混雑しているときは、要請ボタンを連打することになるかもしれません。せっかちなプレイヤーに有効な戦略は、マップ上で目的地をズームアップし、「Call Transport」ボタンがその真上にくるように配置することです。これにより、別のヘリが空域を離れて枠が空いた瞬間に、素早くクリックして予約を入れることができます。

### 距離とタイミング
ゲームは、LZからの距離に基づいて輸送機の到着時間を計算します。500メートル離れた場所からピックアップを要請した場合、ゲームはプレイヤーが地点に到達するまでの時間を与えるために、ヘリの到着を遅らせます。

| シナリオ | 待ち時間への影響 | 戦術的アドバイス |
| :--- | :--- | :--- |
| **遠くから呼ぶ** | **増加** | 攻撃を受けており、到着に合わせて「ホット」な脱出が必要な場合に使用します。 |
| **LZで呼ぶ** | **最小化** | 最速でピックアップされますが、LZで無防備にさらされる時間が長くなります。 |
| **混雑したサーバー** | **変動** | ヘリが空くまで数分間待たなければならない場合があります。 |

## 新しい脱出ポイントの解放

2026年に旅を始めたばかりの頃、マップの大部分は空白です。最初は自勢力の開始地点付近にある数箇所のLZしか持っていません。島全体に活動範囲を広げるには、実際に新しいエリアへ足を運ぶ必要があります。

タスクや戦利品を求めてマップを探索する際、新しい**Gray Zone Warfareの脱出ポイント**の近くに入ると、それらが自動的にアンロックされます。これらのポイントは、ガソリンスタンド、飛行場、軍事キャンプなどの主要地点（POI）の近くに位置していることが多いです。一度アンロックされると、その勢力でのキャリア期間中はずっと利用可能になり、将来のレイドでその地域に直接展開できるようになります。

### 勢力固有の展開
選択した勢力（ラマン・リカバリー・イニシアチブ、ミトラス・セキュリティ・グループ、またはクリムゾン・シールド・インターナショナル）によって、マップの開始地点が決まります。島の中央部は争奪戦となりますが、ベース近くのLZは一般的に安全です。ただし、フレンドリーファイアは通報対象となる違反行為であることを忘れないでください。LZでの悲劇的な事故を避けるため、マップ上の勢力メンバー（青または緑の四角で表示）を常に把握しておきましょう。

## LZにおける戦術的考慮事項

**Gray Zone Warfareの脱出ポイント**で待機する時間は、ゲーム内で最も危険な瞬間の一つです。ヘリコプターの経路と着陸はエリア内の全員から視認可能で、音も聞こえるため、脱出までの窓口期間中、プレイヤーは「格好の標的」となります。

1.  **遮蔽物を見つける:** 待機中、着陸円の真ん中に立ってはいけません。近くの茂みや構造物の後ろに隠れてください。
2.  **「しゃがみスクロール」テクニック:** `Ctrl + マウスホイール`を使用して、遮蔽物越しに高さを精密に調整し、頭をさらさずに視線を確保します。
3.  **武器の準備:** 武器を適切な射撃モードに設定してください。LZでの近接防御にはフルオートを使用し、遠くから脱出ポイントを狙っているスナイパーを排除する必要がある場合はセミオートに切り替えます。
4.  **音に注意を払う:** キャラクターが咳をしていたり負傷したりしている場合は、ヘリが到着する前に医療品を使用してください。音によって、LZで待ち伏せ（エグジットキャンプ）をしている敵プレイヤーに位置を知られる可能性があります。

> **警告:** LZでの待機中に死亡した場合、死体はすべての戦利品とともにその場に残ります。走って戻って回収を試みることはできますが、2回目に死亡すると、元の死体（およびそのすべての装備）は永遠に消滅します。

## 戦利品と死亡時の仕組み

脱出は、見つけたアイテムを確保する唯一の方法です。「セーフロックボックス」に保管されているものは、脱出に失敗しても保護されます。しかし、主要な武器、アーマー、バックパックの中身は失うリスクがあります。

スクワッドでプレイ中に**Gray Zone Warfareの脱出ポイント**の一つで釘付けになった場合、チームメイトが装備の確保を手伝うことができます。あなたが死亡した場合、友人が死体を右クリックして「Drop All（すべてドロップ）」を選択し、アイテムを地面に置くことができます。これにより、アイテムを回収しに戻る途中で再び死亡しても、装備が消滅するのを防ぐことができます。

| アイテムの種類 | 死亡時の状態 | 脱出成功時の状態 |
| :--- | :--- | :--- |
| **セーフボックス内のアイテム** | **保持される** | ボックス内に残る |
| **装備中の武器** | **死体にドロップされる** | ベースに確保される |
| **クエストアイテム** | **紛失** | 完了/提出可能 |
| **バックパックの戦利品** | **死体にドロップされる** | 売却または保管可能 |

## よくある質問（FAQ）

**Q: なぜ自分の場所にヘリコプターを呼べないのですか？**

A: 主な理由は2つあります。1つ目は、勢力の4機のヘリコプターがすべて使用中の「混雑」状態である可能性。2つ目は、ピックアップ地点として利用するには、発見済みのランディングゾーンの一定範囲内にいる必要があるためです。ジャングルの真ん中のランダムな場所にヘリを呼ぶことはできません。

**Q: 敵プレイヤーは私の脱出ヘリコプターを見ることができますか？**

A: はい。ヘリコプターはゲーム世界内の物理的な実体です。大きな音を立て、遠くからでも視認できます。敵プレイヤーはヘリを撃つこともできますが、ヘリ自体は非常に頑丈です。本当の危険は、あなたが搭乗する瞬間に待ち伏せしようと**Gray Zone Warfareの脱出ポイント**付近で待機しているプレイヤーです。

**Q: どうすればより多くのGray Zone Warfareの脱出ポイントを解放できますか？**

A: マップを探索することで解放されます。ミッションの目的や新しい町に向かって移動すると、新しいLZが発見されたという通知が届きます。その後、そのLZを今後のセッションでの挿入と脱出の両方に使用できるようになります。

**Q: ヘリコプターに乗り遅れたらどうなりますか？**

A: ヘリコプターが着陸し、出発までの猶予期間（通常は約15〜20秒）内に搭乗しなかった場合、ヘリはあなたを置いて出発します。別の輸送を要請するには、そのヘリがマップから消えるまで待つ必要があります。車輪が地面に着いたらすぐに動けるように、常に準備をしておきましょう。
