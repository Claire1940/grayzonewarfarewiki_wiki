## ロックボックス・システムの理解

Gray Zone Warfareのロードアウトにおいて最も重要なコンポーネントの一つが「ロックボックス」です。この特殊なコンテナは、死亡時に最も価値のある資産が略奪されるのを防ぐために設計されています。バックパック、リグ、ポケットとは異なり、ロックボックスの中に入れられたアイテムは、たとえKIA（戦死）してもインベントリに残ります。そのため、高額な通貨、希少な鍵、不可欠なクエストアイテムを確保するための主要なツールとなります。

キャリアを開始した直後は、通常、基本的な「セーフロック」ボックスが支給されます。しかし、ランクを上げ、特定の節目をクリアすることで、より多くのグリッドスペースを持つ大型バージョンを入手できます。

| ロックボックス・ティア | グリッドサイズ | 入手方法 |
| :--- | :--- | :--- |
| **セーフロック** | 2x2 (4スロット) | 標準初期装備 |
| **エリートフォルダー** | 2x3 (6スロット) | 中盤のクエスト報酬 / サポーターエディション |
| **タクティカルケース** | 3x3 (9スロット) | エンドゲームの派閥マイルストーン |

> **💡 ヒント:** 前線作戦基地（FOB）を出発する前に、ウォレット、メディキット、そしてインテルフォルダーや暗号化USBドライブなどの固有のクエストアイテムを必ずロックボックスの中に入れるようにしてください。

## クエスト攻略：Paranoia Agent

「Paranoia Agent」のタスクは、クリムゾン・シールド・インターナショナルの派閥で活動するプレイヤーにとって極めて重要なミッションです。このクエストはゲームの調査的な側面を強調しており、手がかりを追跡し、行方不明者の安否を確認する必要があります。環境との相互作用に基づいて目的が動的に更新されるため、正しい順序に従うことが不可欠です。

### ステップ 1：インテルの特定
最初の目的は、指定されたミッションエリア内にある特定の建物を特定することです。中に入ったら、コンピュータのあるワークステーションを探してください。そのコンピュータの中に、調査中の失踪に関する必要な背景情報が記されたメモがあります。クエストの次の段階に進むには、このメモを読むことが必須です。

### ステップ 2：調査
コンピュータのアイテムメニューを閉じると、マップに二次的な場所が更新されます。密林環境で方向を見失わないよう、コンパスにウェイポイントをマークすることを強くお勧めします。近くにある2階建ての建物に移動する必要があります。

### ステップ 3：遺体の調査
2階建ての構造物に入り、上階へ向かいます。部屋の右隅に移動すると、亡くなった対象者がいます。遺体を調査することで調査フェーズが完了し、FOBに戻って報酬を受け取ることができるようになります。

| 目的の段階 | 場所のタイプ | 必要なアクション |
| :--- | :--- | :--- |
| **フェーズ 1** | オフィスビル | コンピュータの**メモを読む** |
| **フェーズ 2** | 2階建ての家 | 2階で**遺体を調査する** |
| **フェーズ 3** | ベースキャンプ | ベンダーに**タスクを報告する** |

## 医療システムとフィールドでの回復

2026年時点での医療システムは非常に詳細です。プレイヤーには単なる「体力バー」があるわけではなく、失血、臓器損傷、骨折などを管理しなければなりません。間違った医療アイテムを使用すると、リソースの無駄遣いになるだけでなく、最悪の場合、フィールドでの緩やかな死を招くことになります。

| 負傷タイプ | 症状 | 推奨される治療 |
| :--- | :--- | :--- |
| **軽度の出血** | 緩やかな体力の減少 | **包帯**（滅菌済み） |
| **重度の出血** | 急速な失血 | **止血帯** / クイッククロット |
| **骨折** | 移動の制限 | **副木** / サージカルキット |
| **臓器損傷** | 咳 / 痛み | **鎮痛剤** ＆ 脱出 |

> **警告:** 止血帯を使用すると重度の出血を即座に止めることができますが、安全圏に到達した後に恒久的なドレッシングと交換しなければ、長期的な肢体損傷を引き起こす可能性があります。

## ラマン島でのサバイバル戦略

環境を真にマスターするには、基本的なチュートリアル以上の知識が必要です。Gray Zone WarfareのAIは、一つの場所に長く留まりすぎるプレイヤーを側面から攻撃し、制圧するように設計されています。

*   **射撃後は移動する:** 射撃して仕留められなかった場合、自分の位置が露呈したと想定してください。少なくとも20〜30メートル移動して、新しい角度を確保しましょう。
*   **水分補給の管理:** ラマン島の熱帯気候は急速な脱水症状を引き起こします。20分以上かかるレイドでは、必ず少なくとも2本のボトル入り飲料水を持ち歩いてください。

## FAQ

**Q: Gray Zone Warfareでロックボックスをアップグレードするにはどうすればよいですか？**

A: 高レベルのトレーダークエストを完了するか、ゲームの上位エディションを購入することでロックボックスをアップグレードできます。2026年の季節イベントによっては、一時的または永続的なストレージ拡張が提供されることもあります。

**Q: クエストアイテムをロックボックスに入れ忘れたらどうなりますか？**

A: ポケットやバックパックにクエストアイテムを入れたまま死亡すると、遺体と一緒にドロップされます。死体まで戻って回収するか、場合によってはクエストの目的を最初からやり直す必要があります。ミッションを開始する前に**gray zone wiki**を確認し、どのアイテムを保護すべきか特定しておくことが役立ちます。

**Q: 他のプレイヤーにロックボックスの中身を盗まれることはありますか？**

A: いいえ。ロックボックスは安全なコンテナであり、他のプレイヤーが略奪することはできません。フィールドで敗北した際にリスクにさらされるのは、外部インベントリ（ベスト、バックパック、ポケット）にあるアイテムのみです。

**Q: Paranoia Agentクエストの遺体はどこにありますか？**

A: 遺体は最初のインテル入手場所の近くにある建物の2階にあります。2階建ての構造物を探し、上階の右隅の部屋を確認して目的を達成してください。
