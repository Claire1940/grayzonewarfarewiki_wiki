## GZW Livestock：タスクの概要と要件

現場に向かう前に、Bansheeが何を期待しているかを理解することが重要です。このタスクは、違法な「家畜（livestock）」輸送の報告を調査することに焦点を当てていますが、それは単なる畜産よりもはるかに邪悪なものであることが判明します。通行量の多いエリアで活動することになるため、適切な装備を整えて臨むことは必須です。

| 要件 | 詳細 |
| :--- | :--- |
| **ベンダー** | **Banshee** |
| **場所** | 自動車修理工場の南西 / 北の家 |
| **目的** | ケージの特定、オフィスコードの発見、機密情報の回収 |
| **推奨レベル** | 15以上（Spearheadアップデート） |
| **必要なアイテム** | ランダムなコード（ミッション内で発見） |

> **💡 ヒント：** ミッションの最後に回収する機密書類のために、インベントリに少なくとも2つの空きスロット（2x1）があることを確認してください。

## ステップ1：ケージと木造小屋を特定する

**gzw livestock**ミッションの最初の段階は、指定された目的サークルの南西の端から始まります。特定の木造小屋と、そこに「家畜」が収容されていたことを示す一連のケージを探します。

1.  **自動車修理工場へ向かう**: 村の工業地帯を目指します。
2.  **南西へ移動**: 自動車修理工場エリアの左下の角へ移動します。
3.  **小屋を特定**: ケージに囲まれた、小さく風化した木造構造物を探します。
4.  **目的を達成**: これらのケージに近づくだけで、メニューの最初のサブタスク完了がトリガーされるはずです。

このエリアは通常2〜3人のAI戦闘員が巡回しています。小屋は飛来する銃弾に対する遮蔽物としてほとんど機能しないため、ケージの近くで無防備に立つ前に、周囲を掃討することをお勧めします。

## ステップ2：自動車修理工場のオフィスコードを見つける

ケージの場所を確認したら、近くのメインオフィスビルがキーパッドまたは南京錠でロックされており、特定のコードが必要であることがわかります。Gray Zone Warfareの他の鍵とは異なり、**gzw livestock**のコードは北にある固定された住宅の中にあります。

| 目的 | 場所の説明 | キーアイテム |
| :--- | :--- | :--- |
| **家を捜索する** | ミッションサークルの北 | **コード付きのメモ** |
| **部屋を特定する** | テレビの近くのリビングルームエリア | **自動車工場のオフィスコード** |
| **コードのバリエーション** | インスタンスごとに固有（例：2896） | **機密情報へのアクセス** |

ケージから北の住宅地へ向かいます。周囲の廃墟に比べて比較的無傷に見える家を探してください。建物に入り、テレビを探します。テレビの近くの小さなテーブルまたはスタンドの上にメモがあります。このメモを調べることで、あなた専用のコードが判明します。

> **警告：** コードはプレイヤーやインスタンスごとにランダム化されます。古いフォーラムの投稿にあるコードを当てにしないでください。進行させるには、現在のセッションで物理的にメモを調べる必要があります。

## ステップ3：オフィスへの侵入と機密情報の回収

コードを入手したら、ケージ近くの最初の目的地点に戻ります。オフィスの入り口は通常、木造小屋に隣接する建物の裏側にあります。

**gzw livestock**タスクを完了するには、以下の手順に従ってください：
*   **南京錠を探す**: オフィスビルの裏口に近づきます。
*   **コードを入力**: 北の家で見つけたコードを使用してドアのロックを解除します。
*   **書類を回収**: 機密情報は通常、小さなオフィススペース内のデスクまたはファイリングキャビネットの上に置いてあります。
*   **アイテムを確保**: 書類をセキュアコンテナ（入る場合）またはバックパックに入れます。

書類を手に入れたら、優先事項は脱出に移ります。この村は、特にミッションに不可欠な建物の近くで、プレイヤー同士の遭遇が発生しやすい「ホットゾーン」として知られています。

## よくある危険とヒント

**gzw livestock**の目的地の周囲は、見かけによらず危険です。AIはまばらに見えるかもしれませんが、彼らはしばしば生い茂った植物や崩れた壁を利用して待ち伏せを仕掛けてきます。

*   **屋根を確認する**: 自動車修理工場には、AIがSKSライフルを持ってスポーンする高所がいくつかあります。ケージに近づく前に、必ず空を確認してください。
*   **「見えない」コード**: テレビの近くにメモが表示されない場合は、再ログインするか、キッチンのカウンターを確認してみてください。まれに、小道具が家具を突き抜けて表示されないことがあります。
*   **分隊の連携**: グループでプレイしている場合は、1人のプレイヤーをオフィスに残し、他の2人がコードを取りに行くようにします。これにより、離れている間にエリアが敵のパトロールに再占拠されるのを防ぐことができます。
*   **脱出経路**: LZ（着陸地点）に戻る際は、大通りを通らないでください。自動車修理工場の西側にある森林地帯を利用して、自分のミッションを開始するために到着した分隊に見つからないようにしましょう。

## FAQ

**Q: ケージに行く前にgzw livestockのコードを見つけることはできますか？**

A: はい、技術的には直接家に向かってコードを入手することも可能ですが、先に南西のケージを訪れるまでミッションログが正しく更新されない場合があります。順番通りに目的を達成するのが安全です。

**Q: 死んで戻ってきた場合、コードは変わりますか？**

A: いいえ、コードは通常、キャラクターの特定のミッションインスタンスに関連付けられています。ただし、別のサーバーや友人の分隊に参加した場合は、コードが変わっていないかメモを再確認する必要があります。

**Q: オフィスの鍵/コードは他のミッションで使用されますか？**

A: 現在、**gzw livestock**タスクで見つかるコードはこの目的専用です。ただし、オフィス自体には、後のレイドでのスカベンジングに役立つマイナーな戦利品スポーンが含まれています。

**Q: 機密書類を紛失した場合はどうなりますか？**

A: 脱出前やセキュアコンテナに入れる前に死亡した場合は、オフィスに戻る必要があります。ドアがすでに解錠されている場合は、そのまま入って再度回収できます。ドアがリセットされている場合は、北の家でコードを再確認する必要があるかもしれません。
