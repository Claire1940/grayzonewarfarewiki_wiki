Gray Zone Warfare の 0.4 Spearhead アップデートでは、プレイヤーをラマン（Lamang）のより危険な地域へと駆り立てる一連の困難な新タスクが導入されました。その中の一つである **gray zone livestock** タスクは、Banshee から割り当てられるミッションで、戦術的な正確さと鋭い観察眼が求められます。このタスクでは、放棄された自動車修理工場を中心に行われている人身売買の凄惨な報告を調査するため、オペレーターをスピールバーグ（Spielberg）地区へと派遣します。**gray zone livestock** ミッションを完了することは、Banshee との評判を高め、2026年のメタにおける上位ティアの装備をアンロックするために不可欠です。

ナム・テュン（Nam Theun）郊外の移動は、最新パッチで AI の密度が増したこともあり、非常に危険です。このガイドでは、証拠の正確な座標、ランダム化されたオフィスコードの場所、そして書類を手に基地へ無事に帰還するための最適な脱出ルートなど、目的のステップバイステップの解説を提供します。

## タスクの概要と要件

現場に向かう前に、ミッションの範囲を理解しておくことが重要です。Banshee は特に違法活動の確実な証拠を探しています。単に敵を掃討するだけでなく、高脅威環境下で調査員として行動することになります。

| タスク要素 | 詳細 |
| :--- | :--- |
| **ベンダー** | **Banshee** |
| **場所** | **スピールバーグ（ナム・テュン エリア）** |
| **主要目標** | 自動車修理工場を調査し、人身売買の証拠を見つける |
| **副次目標** | オフィスのコードを見つけ、書類を回収する |
| **推奨 LZ** | **November 1 または November 2** |
| **脅威レベル** | **中〜高** |

> **警告：** スピールバーグ地区は、周囲の茂みに潜む AI スナイパーで悪名高い場所です。ヘルメットの防弾性能を確認し、少なくとも2つのサージカルキット（外科用キット）を携帯してください。

## ステップ 1：自動車修理工場の特定

**gray zone livestock** タスクの最初のフェーズでは、スピールバーグ地域に飛ぶ必要があります。多くのプレイヤーは **LZ November 2** に着陸するのが最も直接的なアプローチであると感じていますが、**November 1** は森を抜けるより隠密性の高い側面ルートを提供します。

到着したら、放棄された自動車修理工場に向かってください。この建物がミッションの中心となります。最初の目的は、人身売買の物理的な証拠を見つけることです。この目的を完了させるには、屋外のケージ（檻）を見つける必要があります。

### 証拠の発見
ケージは、メインの修理工場建物の左下隅（南西）にある小さな木造の小屋/構造物の中にあります。ここで特定のアイテムと「インタラクト」する必要は必ずしもありません。通常、ケージの鉄格子に近づくか触れることで、サブタスクの完了がトリガーされます。

| 目的の場所 | 座標 / 説明 |
| :--- | :--- |
| **人身売買のケージ** | 修理工場敷地内の南西の角 |
| **メインオフィス** | 修理工場建物の裏側に位置 |
| **コードの家** | 修理工場の北、座標 **182 127** |

## ステップ 2：オフィスのコードを見つける

ケージの存在を確認すると、**gray zone livestock** タスクが更新され、工場の裏手にあるロックされたオフィスに入るよう求められます。このドアはデジタル南京錠で固定されており、爆薬やロックピックで突破することはできません。現地に残された特定のコードを見つける必要があります。

修理工場内でコードを探して時間を無駄にしないでください。代わりに、修理工場から北にある住宅密集地へ向かいます。座標 **182 127** 付近にある特定の家を探してください。

1.  修理工場から北へ移動する。
2.  テレビの目の前にある家に入る。
3.  テーブルの上やテレビの近くにある小さなメモや紙切れを探す。
4.  **注意：** このコードはプレイヤーごとにランダム化されています。よく見られるコードには **2896** や **2381** がありますが、自分のセッションで正しいコードを確認するために、必ず自分でメモをチェックしてください。

## ステップ 3：書類の回収

コードを覚えたら（または書き留めたら）、自動車修理工場に戻ります。建物の裏手に向かうと、キーパッド付きのドアがあります。

### オフィスへの入室
南京錠にインタラクトし、北の家で見つけた4桁のコードを入力します。中に入ると、オフィスは比較的狭いです。書類の山やフォルダを探してください。これが Banshee が要求している証拠です。

> **💡 ヒント：** 書類を拾うと、クエスト用インベントリのスロットを占有します。脱出前に死亡した場合、2026年ビルドの現在の勢力の永続性ルールによってはアイテムを失う可能性があります。書類を手に入れたら、すぐに LZ へ向かうことを強くお勧めします。

| 回収するアイテム | 外観の説明 |
| :--- | :--- |
| **人身売買の台帳** | デスクの上にある黄色/茶色のフォルダ |
| **従業員記録** | ファイルキャビネットの近くにあることが多い |
| **重要書類** | 特に「Livestock」タスク用にラベル付けされている |

## スピールバーグでの戦術的アドバイス

**gray zone livestock** ミッションが行われるエリアは視界が広く、AI にとっての隠れ場所も豊富です。スピールバーグは、長時間の銃撃戦を挑むべき場所ではありません。

### 推奨装備
*   **武器：** 4倍または6倍の光学サイトを備えたアサルトライフルが理想的です。コードの家と修理工場の間の距離は、レッドドットサイトでは不十分と感じるほどです。
*   **弾薬：** 徹甲弾（AP）を使用してください。0.4 アップデートの現地民兵は、レベル II または III のソフトアーマーを着用していることが多いです。
*   **ユーティリティ：** スモークグレネードは、修理工場と北の家の間の道路を横切る際に非常に役立ちます。AI は開けたアスファルトを移動するプレイヤーを追跡してくるからです。

### 脱出戦略
最寄りの脱出ポイントは通常、到着した場所（November 2）です。しかし、N2 で激しい銃声が聞こえたり、プレイヤーのアクティビティが高い場合は、東の川に向かって歩き、予備の LZ へ回り込むことを検討してください。書類が最優先事項です。目的を完了した後は、不要な交戦を避けてください。

## 手順のまとめ

**gray zone livestock** タスクを成功させるために、以下のチェックリストに従ってください。

1.  ベースキャンプで **Banshee** からタスクを受ける。
2.  **LZ November 2** にインフィルする。
3.  **自動車修理工場**へ向かい、木造小屋の中にあるケージに近づく。
4.  北にある **182 127** の家へ移動し、テレビの横にあるコードを読む。
5.  修理工場に戻り、オフィスの南京錠にコードを入力する。
6.  デスクから書類を回収する。
7.  無事に脱出し、Banshee に書類を渡す。

最新の 0.4 Spearhead アップデートの詳細については、[Gray Zone Warfare 公式 Steam ページ](https://store.steampowered.com/app/2477940/Gray_Zone_Warfare/)でパッチノートやコミュニティの議論を確認してください。

## よくある質問（FAQ）

**Q: gray zone livestock タスクのオフィスコードは全員共通ですか？**

A: いいえ。コードはランダム化されています。2896 や 2381 といったコードが報告されていますが、自分のセッション用に生成された特定のコードを確認するには、座標 182 127 の家を訪れる必要があります。

**Q: ケージに到達しましたが、タスクが更新されません。どうすればよいですか？**

A: ケージの判定エリアが狭い場合があります。小さな小屋の中に入り、ケージの扉に直接触れるように立ってみてください。メインの修理工場の南西にある木造構造物であることを確認してください。

**Q: このタスクは分隊で完了できますか？**

A: はい。ただし、分隊の各メンバーがオフィスから自分の書類セットを拾う必要があります。ただし、コードは同じインスタンス内の同じ分隊メンバーであれば共通であるはずです。

**Q: コードを入手した後、書類を手に入れる前に死んでしまった場合はどうなりますか？**

A: コードを覚えているか書き留めていれば、通常は再度探しに行く必要はありません。次回の出撃時に直接オフィスの南京錠に向かうことができます。ただし、書類を拾った後に死亡した場合は、再度回収し直す必要があります。
