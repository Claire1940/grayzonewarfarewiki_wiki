## クエストの概要と目的

「Nicked Shipment」クエストは通常、主要な武器・弾薬ベンダーであるGunnyから発行されます。彼は自分の陣営の開始地点にある町のガソリンスタンド付近で、補給品の貨物を紛失しました。あなたの仕事は、特定の車両を見つけ、中身を確認し、報告することです。

| 目的 | 場所 | 必要条件 |
| :--- | :--- | :--- |
| **貨物を特定する** | 地元のガソリンスタンド | 白い配送トラックを見つける |
| **貨物を捜索する** | トラック内部 | 貨物を調べる（インタラクト） |
| **安全に脱出する** | 最寄りのLZ | 基地への帰還まで生存する |

> **💡 ヒント:** 出発前に、「Tasks（タスク）」メニューでタスクを承諾（Accept）していることを確認してください。「Accept」をクリックし忘れると、現地で目標物とインタラクトすることができません。

## ステップバイステップ・ウォークスルー

### 1. ガソリンスタンドへの移動
自分の陣営の開始地点にある町に最も近い着陸地点（LZ）に展開します。着陸したら、ガソリンスタンドに向かって移動してください。ガソリンスタンドはAIと敵PMCの両方が頻繁に行き交う場所なので、注意が必要です。

### 2. 周辺の掃討
AIのガードが給油ポンプやコンビニエンスストアの周囲をパトロールしていることが多いです。周囲の壁や車両を遮蔽物として利用しましょう。M870を使用している場合は、最大限の効果を発揮するためにAIが15～20メートル以内に近づくのを待ちます。スコープ付きのサブ武器を持っていない限り、遠距離での撃ち合いは避けましょう。

### 3. 白いトラックの特定
貨物はガソリンスタンドの近くに停まっている白い配送トラックの荷台にあります。
*   トラックの後部に近づきます。
*   「Search（捜索）」または「Inspect（調査）」のプロンプトが出るのを確認します。
*   UIのタスクステータスが更新されたら、離脱の準備完了です。

### 4. 回復とメンテナンス
戦闘中にダメージを受けた場合は、すぐにメディカルキットを使用してください。2026年の医療システムでは、傷の種類に応じて特定のアイテムが必要になります：

| 傷の種類 | 必要な医療アイテム |
| :--- | :--- |
| **重度の出血** | **CT (止血帯)** |
| **中程度/軽度の傷** | **包帯** |
| **骨折** | **副木** |
| **臓器損傷** | **Surikit (手術キット)** |

> **⚠️ 警告:** 臓器損傷を無視しないでください。健康メニューで肺や胃が「黄色」または「赤色」になっている場合、スタミナの回復が著しく低下し、脱出が困難になります。

## 隠しベンダー「Vulture」の解放

**gray zone warfare nicked shipment quest**のようなクエストを完了すると、Gunnyの評判が上がるだけでなく、隠しベンダーである「Vulture（ヴァルチャー）」を発見する道も開かれます。Vultureは0.4アップデートで追加されたベンダーで、死亡したPMCから回収された装備の転売を専門としています。

Vultureは一箇所に留まりません。マップ上に3つの黒い点でマークされた「主要COP（拠点）」の間を毎週移動します。主な場所には**Harrison**や**Titan**が含まれます。彼の在庫にアクセスするには、あなたの陣営が現在、彼が駐留しているCOPを支配している必要があります。

### なぜVultureのレベルを上げるのか？
Gunnyが標準的な弾薬を提供するのに対し、Vultureは忠誠度ティアが上がると価値の高い収納ソリューションを提供してくれます：
*   **レベル 3:** 小型セーフティボックスの解放。
*   **レベル 4:** 大型セーフティボックス（現金や高価値アイテム用の108スロット）の解放。

彼のレベルを上げるには、特定の軍事装備や、Midnight Sapphireなどで見つかる高価値アイテムを彼に売却する必要があります。

## Nicked Shipment クエストの高度な戦術

人口の多いPvPサーバーで**gray zone warfare nicked shipment quest**に苦戦している場合は、以下の高度な戦略を検討してください：

1.  **夜間作戦:** 暗闇を隠れ蓑にしましょう。AIの検知範囲がわずかに減少し、暗視ゴーグル（NVG）を持っていないプレイヤーはトラックの近くにいるあなたを見つけにくくなります。
2.  **フランク（側面攻撃）:** ほとんどのプレイヤーはメイン道路からガソリンスタンドに近づきます。森や裏路地を回り込んで、AI（およびプレイヤー）の不意を突いてみましょう。

4.  **鍵の略奪:** ガソリンスタンドにいる間、倒したすべてのAIのポケットを確認してください。彼らは近隣の家や保管室の鍵を持っていることが多く、そこには高価値の戦利品があったり、その後のクエストで必要になったりします。

## FAQ

**Q: gray zone warfare nicked shipment quest のトラックは正確にはどこにありますか？**

A: トラックは、自分の陣営の開始地点の町（Nam Thaven、Khoa、または Mithras）にあるガソリンスタンドに停まっている白い配送車両です。通常、メイン建物の側面または後部付近にあります。

**Q: このクエストは分隊で完了できますか？**

A: はい。分隊全員がクエストを承諾していれば、全員がトラックとインタラクトしてクレジットを受け取ることができます。ただし、ミッションを完了するには全員が安全に脱出する必要があります。

**Q: 貨物を見つけた後に死んだらどうなりますか？**

A: 目的が「特定（Locate）」または「調査（Inspect）」だった場合、通常は進行状況が保存されます。ただし、拾い上げる必要のある物理的なアイテムがあった場合、死亡時にそれを失うため、再びガソリンスタンドに戻って回収する必要があります。

**Q: このミッションには本当にM870が最適ですか？**

A: 2026年バージョン0.4のメタでは、M870は開始地点の町において最もコスト効率の良いツールです。高価な5.56mm弾を節約しつつ、アーマーを着ていないAIに対して信頼性の高い阻止力を発揮します。
