La mise à jour 0.4 Spearhead pour Gray Zone Warfare a introduit une série de nouvelles tâches stimulantes qui poussent les joueurs vers les territoires les plus dangereux de Lamang. Parmi celles-ci se trouve la tâche **Livestock**, une mission confiée par Banshee qui exige une précision tactique et un sens aigu de l'observation. Cette tâche envoie les opérateurs dans la zone de Spielberg pour enquêter sur des rapports sinistres de trafic d'êtres humains centrés autour d'un atelier de réparation automobile abandonné. Terminer la mission **Livestock** est essentiel pour les joueurs souhaitant augmenter leur réputation auprès de Banshee et débloquer de l'équipement de haut niveau dans la méta de 2026.

Naviguer dans la périphérie de Nam Theun peut être périlleux, surtout avec la densité accrue de l'IA introduite dans le dernier patch. Ce guide fournit une décomposition étape par étape des objectifs, incluant les coordonnées exactes des preuves, l'emplacement du code de bureau aléatoire et les meilleurs itinéraires d'extraction pour garantir votre retour à la base avec les documents en main.

## Aperçu de la tâche et exigences

Avant de partir sur le terrain, il est vital de comprendre la portée de la mission. Banshee recherche spécifiquement des preuves tangibles d'activités illégales. Vous ne ferez pas que nettoyer la zone des hostiles ; vous agirez en tant qu'enquêteur dans un environnement à haut risque.

| Élément de la tâche | Détails |
| :--- | :--- |
| **Marchand** | **Banshee** |
| **Lieu** | **Spielberg (zone de Nam Theun)** |
| **Objectif principal** | Inspecter l'atelier de réparation pour des preuves de trafic |
| **Objectif secondaire** | Trouver le code du bureau et récupérer les documents |
| **LZ recommandée** | **November 1 ou November 2** |
| **Niveau de menace** | **Moyen-Élevé** |

> **Avertissement :** La zone de Spielberg est réputée pour ses tireurs d'élite de l'IA cachés dans la végétation environnante. Assurez-vous que votre casque possède un indice balistique élevé et emportez au moins deux kits chirurgicaux.

## Étape 1 : Localiser l'atelier de réparation automobile

La première phase de la tâche **Livestock** nécessite de s'envoler vers la région de Spielberg. La plupart des joueurs trouvent que l'atterrissage à la **LZ November 2** offre l'approche la plus directe, bien que **November 1** propose un itinéraire de flanc plus discret à travers les bois.

Une fois arrivé, dirigez-vous vers l'atelier de réparation automobile abandonné. Cette structure sert de point central pour la mission. Votre premier objectif est de trouver une preuve physique de trafic d'êtres humains. Pour valider cet objectif, vous devez localiser les cages extérieures.

### Trouver les preuves
Les cages sont situées dans une petite cabane ou structure en bois au coin inférieur gauche (sud-ouest) du bâtiment principal de l'atelier de réparation. Vous n'avez pas nécessairement besoin d'interagir avec un objet spécifique ici ; généralement, s'approcher ou toucher les barreaux métalliques des cages déclenchera la validation de l'objectif.

| Emplacement de l'objectif | Coordonnées / Description |
| :--- | :--- |
| **Cages de trafic** | Coin sud-ouest du complexe de l'atelier |
| **Bureau principal** | Situé à l'arrière du bâtiment de l'atelier |
| **Maison du code** | Au nord de l'atelier, à **182 127** |

## Étape 2 : Trouver le code du bureau

Après avoir confirmé la présence des cages, la tâche **Livestock** se met à jour, vous demandant d'entrer dans le bureau verrouillé à l'arrière de l'atelier. Cette porte est sécurisée par un cadenas numérique, et vous ne pouvez pas la forcer avec des explosifs ou des crochets. Vous devez trouver le code spécifique laissé par les opérateurs.

Ne perdez pas de temps à chercher le code dans l'atelier. Dirigez-vous plutôt vers le nord de l'atelier en direction du groupe de maisons résidentielles. Vous cherchez une maison spécifique située approximativement aux coordonnées **182 127**.

1.  Déplacez-vous vers le nord depuis l'atelier de réparation.
2.  Entrez dans la maison située directement devant le téléviseur.
3.  Cherchez une petite note ou un morceau de papier posé sur une table ou près de la télé.
4.  **Note :** Ce code est aléatoire pour chaque joueur. Les codes courants observés sur le terrain incluent **2896** ou **2381**, mais vous devez vérifier la note vous-même pour vous assurer d'avoir le bon code pour votre session.

## Étape 3 : Récupérer les documents

Une fois le code mémorisé (ou noté), retournez à l'atelier de réparation. Allez à l'arrière du bâtiment où vous trouverez une porte avec un pavé numérique.

### Entrer dans le bureau
Interagissez avec le cadenas et entrez le code à 4 chiffres que vous avez trouvé dans la maison au nord. Une fois à l'intérieur, le bureau est relativement petit. Cherchez une pile de papiers ou un dossier — c'est la preuve dont Banshee a besoin.

> **💡 Conseil :** Une fois les documents ramassés, ils occupent un emplacement dans votre inventaire de quête. Si vous êtes tué avant l'extraction, vous risquez de perdre l'objet selon les règles de persistance de votre faction dans la version 2026. Il est fortement recommandé de se diriger directement vers la LZ une fois les papiers en votre possession.

| Objet à récupérer | Description visuelle |
| :--- | :--- |
| **Registre de trafic** | Un dossier jaune/beige sur le bureau |
| **Dossiers des employés** | Souvent trouvés près du classeur |
| **Le document clé** | Spécifiquement étiqueté pour la tâche "Livestock" |

## Conseils tactiques pour Spielberg

La mission **Livestock** se déroule dans une zone offrant de longues lignes de vue et de nombreuses cachettes pour l'IA. Spielberg n'est pas un endroit où vous voulez vous engager dans une fusillade prolongée.

### Équipement recommandé
*   **Arme :** Un fusil d'assaut avec une optique 4x ou 6x est idéal. Les distances entre la maison du code et l'atelier sont juste assez longues pour que les points rouges paraissent insuffisants.
*   **Munitions :** Utilisez des munitions perforantes (AP). La milice locale dans la mise à jour 0.4 porte souvent des protections souples de niveau II ou III.
*   **Utilitaires :** Les grenades fumigènes sont incroyablement utiles pour traverser la route entre l'atelier et les maisons au nord, car l'IA traque souvent les joueurs sur l'asphalte à découvert.

### Stratégie d'extraction
Le point d'extraction le plus proche est généralement celui par lequel vous êtes arrivé (November 2). Cependant, si vous entendez des tirs nourris ou voyez une forte activité de joueurs à N2, envisagez de marcher vers l'est en direction de la rivière et de faire un détour vers une LZ secondaire. Les documents sont la priorité ; évitez les engagements inutiles une fois l'objectif terminé.

## Résumé des étapes

Pour garantir le succès de la tâche **Livestock**, suivez cette liste de contrôle condensée :

1.  Acceptez la tâche auprès de **Banshee** au camp de base.
2.  Infiltrez-vous à la **LZ November 2**.
3.  Rendez-vous à l'**atelier de réparation automobile** et approchez-vous des cages dans la cabane en bois.
4.  Voyagez vers le nord jusqu'à la maison à **182 127** et lisez le code près de la télé.
5.  Retournez à l'atelier, entrez le code dans le cadenas du bureau.
6.  Récupérez les documents sur le bureau.
7.  Extrayez-vous en toute sécurité et remettez les documents à Banshee.

Pour plus d'informations sur les dernières mises à jour 0.4 Spearhead, visitez la [page Steam officielle de Gray Zone Warfare](https://store.steampowered.com/app/2477940/Gray_Zone_Warfare/) pour les notes de patch et les discussions de la communauté.

## FAQ

**Q : Le code du bureau est-il le même pour tout le monde dans la tâche Livestock ?**

R : Non. Le code est aléatoire. Bien que certains joueurs rapportent des codes comme 2896 ou 2381, vous devez visiter la maison aux coordonnées 182 127 pour trouver le code spécifique généré pour votre session.

**Q : J'ai atteint les cages mais la tâche ne s'est pas mise à jour. Que dois-je faire ?**

R : La zone de déclenchement pour les cages peut être étroite. Essayez d'entrer à l'intérieur de la petite cabane et de vous tenir directement contre les portes des cages. Assurez-vous d'être à la structure en bois au sud-ouest du bâtiment principal de réparation.

**Q : Puis-je accomplir cette tâche en escouade ?**

R : Oui, mais chaque membre de l'escouade doit ramasser son propre ensemble de documents dans le bureau. Le code, cependant, devrait être le même pour tous les membres de la même escouade dans cette instance.

**Q : Que se passe-t-il si je meurs après avoir obtenu le code mais avant d'avoir les documents ?**

R : Vous n'avez généralement pas besoin de retrouver le code si vous vous en souvenez ou si vous l'avez noté. Vous pouvez retourner directement au cadenas du bureau lors de votre prochain déploiement. Cependant, si vous mourez après avoir ramassé les documents, vous devrez probablement les récupérer à nouveau.
