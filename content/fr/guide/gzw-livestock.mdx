Terminer la tâche **gzw livestock** est une étape cruciale pour les joueurs cherchant à augmenter leur réputation auprès de Banshee dans la dernière mise à jour 0.4 de Gray Zone Warfare. Cette mission nécessite un mélange de reconnaissance, d'exploration et d'extraction tactique alors que vous naviguez dans un village hostile pour découvrir de sombres secrets concernant l'activité locale. Comme de nombreuses tâches de la mise à jour Spearhead, la mission **gzw livestock** récompense les joueurs qui prêtent une attention particulière aux indices environnementaux et maintiennent un haut niveau de conscience situationnelle.

Dans ce guide, nous détaillerons chaque objectif de la mission, de la découverte initiale des cages de détention à la récupération du code crypté du bureau de l'atelier. Que vous jouiez en solo ou avec une escouade, comprendre la disposition de l'atelier de réparation automobile et des maisons résidentielles environnantes est vital pour votre survie. Suivez attentivement ces étapes pour vous assurer de sécuriser les renseignements nécessaires et de revenir à la base en un seul morceau.

## GZW Livestock : Aperçu de la tâche et conditions

Avant de partir sur le terrain, il est important de comprendre ce que Banshee attend de vous. Cette tâche est centrée sur l'enquête suite à des rapports de transport illégal de « bétail » (livestock), qui s'avère être bien plus sinistre qu'un simple élevage d'animaux. Vous opérerez dans une zone à fort trafic, il est donc non négociable de venir préparé avec le bon équipement.

| Condition | Détail |
| :--- | :--- |
| **Vendeur** | **Banshee** |
| **Emplacement** | Sud-ouest de l'atelier de réparation / Maison au nord |
| **Objectifs** | Localiser les cages, trouver le code du bureau, récupérer les infos |
| **Niveau recommandé** | 15+ (Mise à jour Spearhead) |
| **Objet clé requis** | Code aléatoire (trouvé en mission) |

> **💡 Conseil :** Assurez-vous d'avoir au moins deux emplacements d'inventaire vides (2x1) pour le document de renseignement que vous devez récupérer à la fin de la mission.

## Étape 1 : Localiser les cages et la cabane en bois

La première phase de la mission **gzw livestock** commence à la bordure sud-ouest du cercle d'objectif désigné. Vous recherchez une cabane en bois spécifique et une série de cages qui indiquent où le « bétail » était détenu.

1.  **Approchez de l'atelier de réparation automobile** : Dirigez-vous vers la zone industrielle du village.
2.  **Naviguez vers le sud-ouest** : Déplacez-vous vers le coin inférieur gauche de la zone de l'atelier.
3.  **Identifiez la cabane** : Cherchez une petite structure en bois usée entourée de cages.
4.  **Déclenchez l'objectif** : Le simple fait de s'approcher de ces cages devrait valider la première sous-tâche dans votre menu.

Cette zone est souvent patrouillée par 2 à 3 combattants de l'IA. Il est recommandé de nettoyer le périmètre avant de rester à découvert près des cages, car la cabane offre très peu de protection solide contre les tirs entrants.

## Étape 2 : Trouver le code du bureau de l'atelier

Une fois que vous avez confirmé l'emplacement des cages, vous constaterez que le bâtiment principal du bureau à proximité est verrouillé par un clavier ou un cadenas qui nécessite un code spécifique. Contrairement à certaines clés dans Gray Zone Warfare qui peuvent être pillées sur n'importe quelle IA, le code de **gzw livestock** se trouve dans un bâtiment résidentiel fixe au nord.

| Objectif | Description de l'emplacement | Objet clé |
| :--- | :--- | :--- |
| **Fouiller la maison** | Au nord du cercle de mission | **Note avec le code** |
| **Identifier la pièce** | Salon, près du téléviseur | **Code du bureau** |
| **Variation du code** | Unique par instance (ex: 2896) | **Accès aux infos** |

Dirigez-vous vers le nord depuis les cages vers les maisons résidentielles. Vous cherchez une maison qui semble relativement intacte par rapport aux ruines environnantes. Entrez dans le bâtiment et localisez le téléviseur. Sur une petite table ou un support près de la TV, vous trouverez une note. Interagir avec cette note révélera votre code spécifique.

> **Attention :** Le code est aléatoire pour chaque joueur ou instance. Ne vous fiez pas aux codes trouvés sur d'anciens forums ; vous devez physiquement interagir avec la note dans votre session actuelle pour progresser.

## Étape 3 : Entrer dans le bureau et extraire les renseignements

Une fois le code en main, retournez sur le site d'origine près des cages. L'entrée du bureau se trouve généralement à l'arrière du bâtiment adjacent à la cabane en bois.

Suivez ces étapes pour terminer la tâche **gzw livestock** :
*   **Localisez le cadenas** : Approchez de la porte arrière du bâtiment du bureau.
*   **Entrez le code** : Utilisez le code trouvé dans la maison au nord pour déverrouiller la porte.
*   **Saisissez les documents** : Les renseignements sont généralement posés sur un bureau ou un classeur à l'intérieur du petit espace de bureau.
*   **Sécurisez l'objet** : Placez le document dans votre conteneur sécurisé (s'il y a de la place) ou votre sac à dos.

Une fois le document en votre possession, votre priorité devient l'extraction. Le village est une « zone chaude » réputée pour les rencontres PvP, particulièrement près des bâtiments critiques de mission.

## Recommandations d'équipement tactique pour GZW Livestock

Comme cette mission nécessite de se déplacer entre deux endroits différents (la cabane et la maison au nord), vous êtes exposé à plusieurs lignes de mire. Nous recommandons un équipement équilibré favorisant les engagements à moyenne portée.

| Catégorie d'équipement | Article recommandé | Raison |
| :--- | :--- | :--- |
| **Arme principale** | **M4A1 ou AK-74** | Polyvalence dans les rues du village et le nettoyage de maisons. |

| **Armure** | **Porte-plaques Classe 3+** | Protège contre les munitions 5.45 et 7.62 courantes chez l'IA. |
| **Médical** | **2x Attelles, 4x Pansements** | Forte probabilité de blessures aux membres pendant le trajet vers le nord. |

## Dangers courants et conseils

La zone entourant les objectifs de **gzw livestock** est trompeusement dangereuse. Bien que l'IA puisse sembler rare, elle utilise souvent le feuillage dense et les murs délabrés pour tendre des embuscades.

*   **Vérifiez les toits** : L'atelier de réparation dispose de plusieurs positions surélevées où l'IA peut apparaître avec des fusils SKS. Scannez toujours l'horizon avant d'approcher des cages.
*   **Le code « invisible »** : Si la note n'apparaît pas près de la TV, essayez de vous reconnecter ou de vérifier le comptoir de la cuisine. Dans de rares cas, l'objet peut glitcher à travers le mobilier.
*   **Coordination d'escouade** : Si vous jouez en groupe, demandez à un joueur de rester au bureau pendant que les deux autres vont chercher le code. Cela empêche la zone d'être réoccupée par des patrouilles ennemies pendant votre absence.
*   **Chemin d'extraction** : Ne prenez pas la route principale pour retourner à la zone d'atterrissage (LZ). Utilisez la lisière de la forêt à l'ouest de l'atelier pour éviter d'être repéré par des escouades arrivant pour leurs propres missions.

## Récompenses de mission

Après une extraction réussie et la remise des renseignements à Banshee, vous recevrez un boost significatif de réputation. C'est essentiel pour débloquer des silencieux de haut niveau et des munitions spécialisées.

*   **Points d'expérience** : 1 200 XP
*   **Réputation du vendeur** : +150 (Banshee)
*   **Devise** : 4 500 $ (environ)
*   **Équipement** : Comprend généralement un ensemble de fournitures médicales rares ou des caisses de munitions.

## FAQ

**Q : Puis-je trouver le code de gzw livestock avant de visiter les cages ?**

R : Oui, vous pouvez techniquement vous rendre directement à la maison pour obtenir le code, mais le journal de mission risque de ne pas se mettre à jour correctement tant que vous n'avez pas visité les cages au sud-ouest en premier. Il est plus sûr de suivre les objectifs dans l'ordre.

**Q : Le code change-t-il si je meurs et que je reviens ?**

R : Non, le code est généralement lié à l'instance de mission spécifique de votre personnage. Cependant, si vous rejoignez un serveur différent ou l'escouade d'un autre ami, vous devriez revérifier la note pour vous assurer que le code n'a pas changé.

**Q : La clé/code du bureau est-il utilisé pour d'autres missions ?**

R : Actuellement, le code trouvé dans la tâche **gzw livestock** est spécifique à cet objectif. Cependant, le bureau lui-même contient quelques butins mineurs qui peuvent être utiles lors de raids ultérieurs.

**Q : Que se passe-t-il si je perds le document de renseignement ?**

R : Si vous mourez avant l'extraction ou avant d'avoir placé le document dans votre conteneur sécurisé, vous devrez retourner au bureau. Si la porte est déjà déverrouillée, vous pouvez simplement entrer et le reprendre. Si la porte s'est réinitialisée, vous devrez peut-être revérifier le code à la maison du nord.
