Naviguer dans les jungles dangereuses et les ruines industrielles de Lamang demande plus qu'une visée stable ; cela requiert de solides compétences d'investigation. La tâche **grey zone warfare livestock**, confiée par le marchand Banshee, est un objectif en plusieurs étapes introduit dans la mise à jour 0.4 qui met les joueurs au défi d'infiltrer un centre de réparation automobile pour récupérer des documents sensibles. Que vous jouiez pour la Lamang Recovery Initiative, la Mithras Security Force ou la Crimson Shield International, accomplir la mission **grey zone warfare livestock** est essentiel pour progresser dans le déblocage de l'équipement de haut niveau de Banshee.

Ce guide propose une analyse complète de l'opération, incluant les emplacements exacts du code secret, du bureau verrouillé et des conseils tactiques pour survivre à la présence d'IA hostiles dans la zone. En suivant ces étapes, vous passerez moins de temps à errer et plus de temps à vous extraire avec les renseignements.

## Aperçu de la tâche et exigences

Avant de partir sur le terrain, il est vital de comprendre ce que Banshee attend de vous. Cette tâche n'est pas une simple mission de « recherche et destruction » ; elle implique une interaction avec l'environnement et la résolution d'une énigme. Vous opérerez dans une zone généralement peuplée de combattants IA de niveau moyen à élevé, il est donc indispensable de venir préparé avec le bon équipement.

| Élément de la tâche | Détails |
| :--- | :--- |
| **Marchand** | **Banshee** |
| **Emplacement** | Atelier de réparation automobile / Zone industrielle |
| **Objectif principal** | Récupérer les renseignements du bureau automobile |
| **Objectif secondaire** | Localiser le code du bureau automobile |
| **Objets requis** | Aucun (Code trouvé en mission) |
| **Difficulté** | Modérée |

> **Attention :** La zone entourant l'atelier de réparation est souvent patrouillée par plusieurs escouades d'IA. Assurez-vous d'avoir au moins deux chargeurs de rechange et des fournitures médicales de base avant de tenter cette tâche.

## Étape 1 : Localiser les cages et la cabane

La première phase de la mission **grey zone warfare livestock** nécessite d'identifier le site d'intérêt principal. Votre destination est une petite cabane en bois située près d'une série de cages pour animaux. Cet emplacement se trouve dans le coin sud-ouest du complexe de l'atelier de réparation automobile.

En arrivant à l'endroit marqué sur votre carte, cherchez la clôture périphérique de l'atelier. Au lieu d'entrer par le portail principal, contournez par le coin inférieur gauche (sud-ouest). Vous verrez une petite cabane en bois usée et plusieurs cages qui semblent avoir été utilisées pour du bétail — d'où le nom de la mission (*Livestock* signifie bétail).

S'approcher de ces cages ou entrer dans la cabane déclenche généralement la validation de la première sous-tâche. Cependant, la porte que vous devez réellement ouvrir est verrouillée par un pavé numérique, ce qui nécessite un détour par une structure résidentielle voisine pour trouver la combinaison.

## Étape 2 : Trouver le code du bureau automobile

Vous ne pouvez pas forcer la porte du bureau ; vous devez trouver la note physique contenant le code. Depuis la cabane en bois et les cages, vous devez vous diriger vers le nord. Cherchez une maison résidentielle située juste à l'extérieur du cercle industriel immédiat.

1.  Sortez de la zone de l'atelier de réparation et déplacez-vous vers le Nord.
2.  Entrez dans la maison (elle possède généralement un téléviseur dans la pièce principale).
3.  Cherchez près du téléviseur une petite note.
4.  Interagissez avec la note pour révéler le code.

Bien que le code ait été rapporté comme étant le **2896** par certains opérateurs, les codes dans Gray Zone Warfare peuvent occasionnellement être aléatoires ou modifiés lors des mises à jour saisonnières. Interagissez toujours avec la note pour vous assurer que l'objectif se met à jour dans votre journal de tâches.

## Étape 3 : Entrer dans le bureau et récupérer les renseignements

Une fois le code en main, retournez à la zone de l'atelier de réparation automobile où vous avez trouvé les cages pour la première fois. Le bureau est situé dans la même structure de bâtiment, mais l'entrée est souvent cachée à l'arrière. Cherchez une porte munie d'un cadenas numérique.

Interagissez avec le cadenas et entrez le code que vous avez découvert dans la maison au nord. Une fois la porte ouverte, entrez dans le bureau et cherchez un bureau ou un classeur. Le document, étiqueté « Car Office Intel », s'y trouvera. Ramassez-le et sécurisez-le immédiatement dans votre « Conteneur sécurisé » si vous avez de la place. Cela garantit que même si vous êtes KIA (tué au combat) pendant l'extraction, vous ne perdrez pas l'objet de mission.

| Composant de l'emplacement | Description | Interaction clé |
| :--- | :--- | :--- |
| **Maison Nord** | Bâtiment résidentiel avec une TV | **Trouver la note/le code** |
| **Cabane Sud-Ouest** | Structure en bois près des cages | **Objectif initial** |
| **Bureau Automobile** | Pièce verrouillée à l'arrière de l'atelier | **Entrer le code & piller l'intel** |

## Conseils tactiques pour la tâche Livestock

La mission **grey zone warfare livestock** se déroule dans une zone offrant de longues lignes de vue et plusieurs positions surélevées où des snipers IA peuvent apparaître. Pour naviguer avec succès dans cette zone en 2026, suivez ces recommandations tactiques :

*   **Les silencieux sont obligatoires :** L'atelier de réparation automobile est une zone « chaude ». L'utilisation d'une arme sans silencieux attirera toutes les IA dans un rayon de 200 mètres vers votre position, rendant la recherche du code nettement plus difficile.
*   **Vérifiez les toits :** Avant d'approcher la maison au nord, observez aux jumelles les toits des bâtiments industriels environnants. Les IA s'y tiennent souvent avec des fusils de type SKS ou AK.
*   **Nettoyez l'intérieur :** Lorsque vous entrez dans la maison pour le code, ne vous précipitez pas. Des IA peuvent parfois apparaître à l'intérieur ou juste derrière la maison. Utilisez une grenade flash si vous soupçonnez une présence ennemie.

> **💡 Conseil :** Si vous jouez en escouade, demandez à un joueur de rester près du bureau pendant que les deux autres récupèrent le code. Cela empêche les IA de réapparaître dans la zone du bureau pendant votre absence.

## Terminer la mission

Une fois les renseignements dans votre inventaire, votre objectif final est de vous extraire en toute sécurité et de remettre les documents à Banshee. Ouvrez votre carte et localisez la zone d'atterrissage (LZ) la plus proche. Soyez prudent, car les LZ proches des centres de mission sont des cibles fréquentes pour les « campeurs de LZ » ou les patrouilles d'IA errantes.

Une fois de retour au camp de base, ouvrez le menu « Marchands », sélectionnez Banshee et accédez à l'onglet « Tâches ». Cliquez sur « Livestock » et sélectionnez « Hand Over » (Remettre) pour terminer la mission.

## FAQ

**Q : Le code pour la tâche Livestock est-il le même pour tout le monde ?**

R : Bien que de nombreux joueurs rapportent le code 2896, Gray Zone Warfare peut parfois rendre les codes aléatoires par serveur ou par instance de joueur dans les nouvelles mises à jour. Il est toujours recommandé de vérifier la note dans la maison nord pour s'assurer que le code est actif pour votre mission spécifique.

**Q : Que se passe-t-il si je meurs après avoir ramassé les renseignements ?**

R : Si les renseignements sont dans votre sac à dos standard ou votre gilet, vous les perdrez à votre mort et devrez retourner à votre corps pour les récupérer. Cependant, si vous placez les renseignements dans votre conteneur sécurisé (Gamma/Epsilon/etc.), ils resteront avec vous même si vous ne parvenez pas à vous extraire.

**Q : Ai-je besoin d'une clé spécifique pour entrer dans la maison avec le code ?**

R : Non, la maison contenant le code est généralement ouverte. Vous n'avez besoin que du code numérique pour la porte finale du bureau où se trouvent les renseignements.

**Q : Puis-je accomplir cette tâche en escouade ?**

R : Oui, mais une seule personne a besoin de trouver le code pour que toute l'escouade puisse l'utiliser. Cependant, chaque joueur doit physiquement ramasser le document de renseignement (ou attendre qu'il réapparaisse) pour valider ses propres exigences de tâche, à moins que la mission ne permette une progression partagée.
