Naviguer dans les jungles denses et les villages dévastés par la guerre de Lamang demande plus qu'une visée stable ; cela nécessite un œil attentif aux détails et la capacité de traquer des renseignements vitaux. Pour les joueurs s'attaquant à la tâche **gray zone warfare livestock**, le défi consiste à découvrir une sombre opération de trafic d'êtres humains cachée dans la région de Spielberg. Cette mission, généralement confiée par le vendeur Banshee suite à la mise à jour Spearhead 0.4, vous emmène au cœur de la zone de Nam Theun, où vous devez enquêter sur un atelier de réparation automobile abandonné et récupérer des documents compromettants.

Terminer la mission **gray zone warfare livestock** est essentiel pour augmenter votre réputation auprès de Banshee et débloquer de l'équipement de niveau supérieur. La tâche se déroule en plusieurs étapes, vous obligeant à trouver des preuves physiques de trafic, à localiser un code d'accès aléatoire dans une zone résidentielle à proximité, et enfin à pénétrer dans un bureau verrouillé pour sécuriser les preuves. Ce guide propose un cheminement pas à pas de toute l'opération, incluant des conseils tactiques pour survivre dans l'environnement à haut risque de Spielberg en 2026.

## Aperçu de la tâche Livestock

Avant de partir sur le terrain, il est important de comprendre les objectifs. La tâche Livestock n'est pas une simple mission de « recherche et destruction ». Elle implique des éléments de reconnaissance et de résolution d'énigmes qui peuvent être mortels si vous n'êtes pas préparé à la présence de la milice locale.

| Objectif | Emplacement | Condition clé |
| :--- | :--- | :--- |
| **Enquêter sur l'atelier de réparation** | Spielberg (Nam Theun) | Atteindre les cages à l'arrière |
| **Localiser le code du bureau** | Maison résidentielle (Nord) | Trouver la note près de la télé |
| **Récupérer les documents** | Bureau de l'atelier | Saisir le code à 4 chiffres |
| **Remettre les preuves** | Camp de base | Retourner voir Banshee |

> **Attention :** La zone de Spielberg est connue pour sa forte densité d'IA. Attendez-vous à de nombreux hostiles patrouillant dans l'atelier de réparation et les rues résidentielles environnantes. Apportez au moins trois chargeurs de rechange et des fournitures médicales adéquates.

## Phase 1 : Localiser l'atelier de réparation automobile

Votre première destination est la zone de Spielberg, plus précisément l'atelier de réparation automobile abandonné situé près de la région de Nam Theun. Le moyen le plus efficace d'atteindre cet endroit est de voler vers la **LZ November 1** ou la **LZ November 2**. Depuis ces zones d'atterrissage, vous devrez vous diriger vers la périphérie sud de la ville.

L'atelier de réparation automobile est un bâtiment industriel distinct. Pour terminer la première partie de la tâche **gray zone warfare livestock**, vous ne devez pas seulement trouver le bâtiment ; vous devez trouver les preuves du trafic d'êtres humains.

1.  Approchez-vous de l'atelier par le sud ou l'ouest pour éviter d'être coincé sur la route principale.
2.  Cherchez une petite cabane en bois ou une zone extérieure contenant des cages en fer.
3.  Vous devez physiquement vous approcher et toucher ces cages pour déclencher la validation de l'objectif « Enquêter ».
4.  Une fois la sous-tâche mise à jour, votre prochain objectif est de trouver le code d'accès du bureau verrouillé à l'intérieur du bâtiment principal.

## Phase 2 : Trouver le code du bureau

Le bureau de l'atelier de réparation est protégé par un cadenas numérique. Contrairement à certaines clés statiques du jeu, le code de la tâche **gray zone warfare livestock** se trouve souvent sur une note physique dans un endroit séparé. Vous devrez vous déplacer légèrement au nord de l'atelier, dans le secteur résidentiel de Spielberg.

La maison spécifique que vous recherchez est située près des coordonnées **182 127**. Cette maison est reconnaissable à sa proximité avec un téléviseur à l'intérieur de la pièce à vivre principale.

*   **Étape 1 :** Dirigez-vous vers le nord depuis l'atelier le long de la route principale.
*   **Étape 2 :** Localisez la maison au 182 127.
*   **Étape 3 :** Entrez dans le bâtiment et vérifiez la table ou le support directement devant la télé.
*   **Étape 4 :** Interagissez avec la note pour lire le code. 

> **Note :** Le code est aléatoire pour les différents joueurs ou sessions. Les codes couramment signalés incluent **2381** ou **2896**, mais vous devez vérifier la note dans votre instance spécifique pour vous assurer que le clavier accepte la saisie.

## Phase 3 : Récupérer les documents

Une fois le code en main, retournez à l'atelier de réparation. Le bureau est situé vers l'arrière de la structure principale. Vous verrez une porte équipée d'un clavier électronique.

| Action | Détails |
| :--- | :--- |
| **S'approcher de la porte** | Située à l'arrière ou sur le côté du bâtiment de l'atelier. |
| **Saisir le code** | Utilisez le code à 4 chiffres trouvé dans la maison résidentielle. |
| **Localiser l'objet** | Les documents sont généralement posés sur un bureau ou une étagère. |
| **Sécuriser l'objet** | Placez immédiatement les documents dans votre **Conteneur sécurisé**. |

Une fois les documents en votre possession, la tâche **gray zone warfare livestock** vous demande de vous extraire en toute sécurité. Comme Spielberg est une zone contestée, le trajet de retour vers la LZ peut être la partie la plus dangereuse de la mission. Si vous mourez avant d'atteindre la LZ, les documents resteront dans votre conteneur sécurisé, mais vous devrez tout de même réussir l'exfiltration pour les remettre à Banshee.

## Recommandations tactiques pour Spielberg

L'IA dans la zone de Nam Theun/Spielberg est nettement plus agressive que dans les villes de départ. Ils portent souvent des armures souples et des fusils de la série AK, ce qui en fait une menace même pour les joueurs de niveau intermédiaire.

1.  **Choix de la LZ :** La LZ November 2 offre une approche légèrement plus discrète à travers la lisière de la forêt, tandis que la November 1 vous place plus près de la route principale où vous êtes facilement repérable.
2.  **Silencieux obligatoires :** Si vous jouez en solo, un silencieux est votre meilleur ami. Les tirs non supprimés attireront l'IA de tout le pâté de maisons vers votre position.
3.  **Vérifiez les toits :** Certains bâtiments de Spielberg ont des toits plats ou des balcons où des tireurs d'élite de l'IA peuvent apparaître. Balayez toujours verticalement avant de traverser des rues dégagées.
4.  **Synergie d'équipe :** Si vous jouez en escouade, demandez à un joueur de surveiller la sécurité à la maison 182 127 pendant qu'un autre récupère le code. Les zones résidentielles comportent de nombreux angles en « L » où l'IA peut se cacher.

## Dépannage des problèmes courants

De nombreux joueurs rencontrent des difficultés lors de la quête **gray zone warfare livestock** en raison de la nature du déclencheur « enquêter ». Si l'objectif ne se valide pas, essayez de marcher autour du périmètre des cages ou de vous tenir à l'intérieur si les portes sont ouvertes.

Un autre problème courant est que la note du code n'apparaît pas. Si la note manque dans la maison au 182 127, assurez-vous d'avoir bien accepté la tâche auprès de Banshee. Dans certains cas, un redémarrage du serveur ou le passage à une autre instance peut être nécessaire si les objets du monde ne s'affichent pas correctement.

| Composant | Problème potentiel | Solution |
| :--- | :--- | :--- |
| **Cages** | L'objectif ne se déclenche pas | Approchez-vous ou allongez-vous près du sol de la cage. |
| **Note du code** | La note n'est pas sur la table | Vérifiez le sol ou redémarrez la session. |
| **Clavier** | Code « Invalide » | Relisez la note ; les codes sont **uniques** par instance. |
| **Extraction** | La LZ est surveillée | Déplacez-vous vers une LZ secondaire plus éloignée. |

## FAQ

**Q : Où se trouve exactement l'atelier de réparation automobile pour la tâche gray zone warfare livestock ?**

R : L'atelier de réparation est situé dans la ville de Spielberg, qui fait partie de la région de Nam Theun. C'est un grand bâtiment industriel sur le côté sud de la ville, identifiable par les cages en fer situées derrière ou à côté.

**Q : Le code du bureau est-il le même pour tout le monde ?**

R : Non, le code du bureau de l'atelier est aléatoire. Bien que certains joueurs aient signalé des codes comme 2381 ou 2896, vous devez trouver la note dans la maison aux coordonnées 182 127 pour obtenir le code correct pour votre instance de mission actuelle.

**Q : Que se passe-t-il si je perds les documents livestock après les avoir ramassés ?**

R : Si vous mourez et que les documents n'étaient pas dans votre conteneur sécurisé, vous devrez retourner au bureau de l'atelier de réparation pour les récupérer à nouveau. Si vous avez déjà utilisé le code, la porte peut rester déverrouillée, ou vous devrez peut-être ressaisir le code.

**Q : Quel vendeur donne la tâche Livestock ?**

R : La tâche Livestock est fournie par Banshee. Elle fait partie de la série de quêtes de la mise à jour 0.4 Spearhead et se concentre sur la découverte des activités illégales des éléments criminels locaux à Lamang.
